## 技术实现

### 前端修改
**文件**: `src/views/QuizView.vue`

```javascript
// 获取设置
const { randomizeAnswers } = settingsStore.settings

// 根据设置决定是否打乱
if (randomizeAnswers) {
  // 打乱选项并生成映射
  const { shuffledOptions, reverseMapping } = shuffleOptions(options);
  return {
    ...q,
    options: shuffledOptions,
    shuffleMapping: reverseMapping
  };
} else {
  // 不打乱，保持原样
  return {
    ...q,
    options: options,
    shuffleMapping: null
  };
}
```

### 后端兼容
**文件**: `routes/quiz.js`

```javascript
// 后端自动适配两种模式
if (shuffleMapping) {
  // 有映射，使用映射验证
  mappedUserAnswer = mapAnswerToOriginal(userAnswer, reverseMapping);
} else {
  // 无映射，直接验证
  mappedUserAnswer = userAnswer;
}
```

### 设置存储
**文件**: `src/stores/questionStore.js`

