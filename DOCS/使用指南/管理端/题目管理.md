# 题目管理

## 进入题目管理页面

1. **登录管理后台**
   - 输入管理员用户名和密码登录系统
   - 进入管理后台首页

2. **进入题目管理**
   - 在左侧导航栏中点击「题目管理」
   - 进入题目管理页面

## 查看题目列表

1. **题目列表展示**
   - 系统会显示所有题目的列表
   - 包含题目ID、学科、题目内容、难度、类型等信息

2. **筛选题目**
   - 可以按学科、难度、类型等条件筛选题目
   - 输入关键词搜索题目内容

3. **分页查看**
   - 题目列表支持分页显示
   - 可以调整每页显示的题目数量

## 添加题目

1. **点击「添加题目」按钮**
   - 在题目管理页面点击「添加题目」按钮
   - 弹出添加题目对话框

2. **填写题目信息**
   - **学科**: 选择题目所属的学科
   - **题目内容**: 输入题目内容
   - **选项**: 输入题目选项（A、B、C、D等）
   - **答案**: 输入正确答案
   - **难度**: 选择题目难度（简单、中等、困难）
   - **类型**: 选择题目类型（单选、多选）

3. **保存题目**
   - 点击「保存」按钮
   - 系统会提示添加成功
   - 新题目会显示在题目列表中

## 编辑题目

1. **选择要编辑的题目**
   - 在题目列表中找到要编辑的题目
   - 点击该题目的「编辑」按钮
   - 弹出编辑题目对话框

2. **修改题目信息**
   - 修改需要更新的字段
   - 可以修改题目内容、选项、答案、难度、类型等

3. **保存修改**
   - 点击「保存」按钮
   - 系统会提示修改成功
   - 题目列表会显示更新后的题目信息

## 删除题目

1. **选择要删除的题目**
   - 在题目列表中找到要删除的题目
   - 点击该题目的「删除」按钮
   - 系统会提示确认删除

2. **确认删除**
   - 点击「确认」按钮
   - 系统会提示删除成功
   - 题目会从列表中移除

3. **批量删除**
   - 可以选择多个题目
   - 点击「批量删除」按钮
   - 系统会提示确认删除
   - 点击「确认」按钮批量删除选中的题目

## 批量导入题目

1. **点击「批量导入」按钮**
   - 在题目管理页面点击「批量导入」按钮
   - 弹出批量导入对话框

2. **准备导入文件**
   - 按照系统要求的格式准备题目文件
   - 支持的格式：文本文件（.txt）
   - 格式示例：
     ```
     1. 题目内容(A)
     A. 选项1
     B. 选项2
     C. 选项3
     D. 选项4

     【多选】多选题(ABC)
     A. 选项1
     B. 选项2
     C. 选项3
     D. 选项4
     ```

3. **上传文件**
   - 点击「选择文件」按钮
   - 选择准备好的题目文件
   - 点击「导入」按钮

4. **导入结果**
   - 系统会显示导入结果
   - 成功导入的题目数量
   - 失败的题目及原因

## 题目预览

1. **预览题目**
   - 在题目列表中点击题目内容
   - 系统会显示题目预览
   - 可以查看题目完整内容和选项

2. **预览答案**
   - 在预览界面可以查看正确答案
   - 可以检查题目设置是否正确

## 注意事项

- **题目内容**: 题目内容应清晰明确，避免歧义
- **选项设置**: 选项应互斥且覆盖所有可能的答案
- **答案设置**: 确保答案正确无误
- **难度设置**: 根据题目实际难度合理设置难度等级
- **批量导入**: 导入前请确保文件格式正确，避免导入失败
- **数据备份**: 定期备份题目数据，防止数据丢失
