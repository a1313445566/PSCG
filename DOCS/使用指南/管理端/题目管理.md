# 题目管理详细操作指南

> PSCG 智能学习系统 - 管理端 | 版本：v1.0 | 最后更新：2026-04-05
> 题目的增删改查、批量操作、导入导出的完整操作手册

---

## 一、功能概述

题目管理是管理后台**最常用也是最重要**的功能模块。它负责维护整个系统的**题库**，所有学生的练习题目都来源于此。

### 1.1 核心能力

```
┌─────────────────────────────────────────────────────┐
│                   题 目 管 理                         │
│                                                     │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐        │
│  │ 新增  │  │ 编辑  │  │ 删除  │  │ 预览  │        │  ← 单题操作
│  └──────┘  └──────┘  └──────┘  └──────┘        │
│                                                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐        │
│  │ 批量导入  │  │ 批量修改  │  │ 批量删除  │        │  ← 批量操作
│  └──────────┘  └──────────┘  └──────────┘        │
│                                                     │
│  ┌──────────────────────────────────────────┐      │
│  │ 搜索 / 筛选 / 排序 / 分页               │      │  ← 查找定位
│  └──────────────────────────────────────────┘      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 1.2 支持的题型

| 题型 | type 值 | 选项格式 | 适用场景 |
|------|---------|----------|----------|
| **单选题** | `single_choice` | A/B/C/D，单选 | 大多数选择题 |
| **多选题** | `multiple_choice` | A/B/C/D/E/F，多选 | 综合选择题 |
| **判断题** | `judgment` | 固定对/错 | 是非判断 |
| **阅读理解** | `reading` | 文章 + 多个小题 | 英语/语文阅读 |

## 三、搜索与筛选

### 3.1 关键词搜索

在搜索框中输入关键词，支持模糊匹配：

- **题目标题**：搜索题目文字内容
- **选项内容**：搜索选项中的文字
- **解析内容**：搜索答案解析

```
示例: 搜索 "集合"
→ 匹配: "设集合 A = {1,2,3}..." 
        "两个集合的交集..."
        "集合的基本运算..."
```

### 3.2 下拉筛选

| 筛选项 | 选项值 | 说明 |
|--------|--------|------|
| **学科** | 全部 / 数学 / 语文 / 英语 / 物理 / 化学 / ... | 按学科过滤 |
| **题型** | 全部 / 单选 / 多选 / 判断 / 阅读 | 按题型过滤 |
| **难度** | 全部 / ⭐ / ⭐⭐ / ⭐⭐⭐ / ⭐⭐⭐⭐ / ⭐⭐⭐⭐⭐ | 按难度过滤 |
| **分类** | 全部 / 各子分类 | 按知识点过滤 |

筛选条件可以**组合使用**，例如：只看"数学"学科中"单选题"且难度为"3星"的题目。

### 3.3 排序

点击表头列名可以排序：

- **ID**: 升序/降序（默认按录入顺序）
- **难度**: 从易到难 / 从难到易
- 其他列排序规则视具体实现而定

### 3.4 分页

默认每页显示一定数量的题目（通常 20 或 50 条），底部显示分页控件：

```
共 1250 条    < 1  2  3  4 ... 50 >
```

- 点击页码跳转
- `<` `>` 上一页/下一页
- 可能在设置中调整每页条数

## 四、新增题目

### 4.1 方式一：表单新增

**步骤**：

```
1. 点击工具栏 "[新增题目]" 按钮
2. 弹出题目编辑对话框/页面
3. 填写各字段（详见下方）
4. 点击"保存"
```

### 4.2 题目表单字段详解

#### 基本信息

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| **学科** | 是 | 下拉选择所属学科（必须先在基础设置中创建学科） |
| **子分类** | 是 | 下拉选择所属知识点/章节 |
| **题型** | 是 | 单选/多选/判断/阅读理解 |
| **难度** | 是 | 1-5 星（影响题目推荐和学生挑战意愿） |

#### 题目内容

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| **题目标题** | 是 | 题目的完整文字内容 |
| **编辑器** | - | 支持**富文本**（Quill 编辑器）：加粗、斜体、公式、列表、颜色等 |

> **提示**：对于含数学公式的题目，使用富文本编辑器的公式功能输入。

#### 选项区域

根据**题型不同**，选项区域的形态也不同：

##### 单选题选项

```
选项 A: [_________________________]   ← 文本输入
选项 B: [_________________________]
选项 C: [_________________________]
选项 D: [_________________________]
正确答案: (○ A  ○ B  ○ C  ○ D)     ← 单选框选择
```

- 输入 4 个选项的内容（A/B/C/D）
- 使用单选框标记**唯一正确答案**

##### 多选题选项

```
选项 A: [_________________________]
选项 B: [_________________________]
选项 C: [_________________________]
选项 D: [_________________________]
选项 E: [_________________________]   ← 可能有更多
正确答案: (☑ A  ☑ C  ☑ E)            ← 多选框（可选择多个）
```

- 输入选项内容（通常 4-6 个）
- 使用**多选框**标记所有正确答案（必须 >= 2 个）

##### 判断题选项

```
判断题无需手动填写选项！
系统自动固定选项为:
  A. 对 (TRUE / 正确)
  B. 错 (FALSE / 错误)

正确答案: (○ A  ○ B)   或   (○ B  ○ A)
```

- **判断题的选项由系统固定**，管理员只需选择哪个是正确的
- 选项内容不可编辑

##### 阅读理解选项

```
阅读材料:
┌──────────────────────────────────────────────┐
│                                              │
│  [富文本编辑器 - 输入阅读文章的完整内容]      │
│  支持长文本、段落、图片                        │
│                                              │
└──────────────────────────────────────────────┘

小题:

小题 1:
  题目: [_______________________________]
  选项 A: [__________]  B: [__________]
  选项 C: [__________]  D: [__________]
  正确答案: ○

小题 2:
  题目: [_______________________________]
  ...

[ + 添加小题 ]   [ - 删除小题 ]
```

- 先输入**阅读材料**（文章全文）
- 然后添加**若干小题**（每道小题可以是单选或多选）
- 每个小题独立设置选项和答案
- 支持动态增减小题数量

#### 附加内容

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| **答案解析** | 否 | 解题思路、详细解释（学生答错后可见） |
| **配图** | 否 | 上传图片文件（jpg/png/gif，最大 10MB） |
| **音频** | 否 | 上传音频文件（mp3/wav 等，最大 10MB） |

##### 上传配图

```
操作步骤:
1. 点击"上传图片"或拖拽图片到上传区域
2. 选择本地图片文件（支持 jpg/png/gif/webp）
3. 等待上传完成（显示进度条）
4. 预览确认图片
5. 图片将作为题目的一部分保存
```

##### 上传音频

```
操作步骤:
1. 点击"上传音频"
2. 选择本地音频文件（mp3/wav/ogg 等）
3. 等待上传完成
4. 可试听确认
5. 音频将在题目中显示播放器
```

### 4.3 保存与取消

| 按钮 | 操作 |
|------|------|
| **保存** | 验证必填字段 → 保存到数据库 → 刷新列表 |
| **取消** | 放弃所有修改 → 关闭表单 |
| **保存并继续** | 保存当前题目后**清空表单**，方便连续添加下一题 |

## 五、编辑题目

### 5.1 行内编辑（适用于简单修改）

```
操作步骤:
1. 在表格中找到目标题目
2. 点击该行的 "✏ 编辑" 图标
3. 该行变为可编辑状态（或弹出小型编辑框）
4. 直接修改要改的字段
5. 点击 "✅ 确定" 保存
```

适用场景：
- 修正个别文字错误
- 修改难度星级
- 更换正确答案

### 5.2 分屏编辑（适用于复杂修改）

```
操作步骤:
1. 双击某行的题目（或点击 "编辑" 按钮）
2. 打开分屏编辑面板:

┌───────────────────┬──────────────────────────────┐
│                   │                               │
│   题目列表         │      题目编辑表单              │
│                   │                               │
│  ✓ 101 数学 单选  │  学科: [数学 ▼]              │
│  ✓ 102 数学 判断  │  题型: [单选 ▼]              │
│   103 英语 阅读  │  难度: [⭐⭐⭐ ▼]            │
│   104 物理 多选  │  题目标题:                    │
│                   │  [________________]           │
│                   │  选项 A: [__________]         │
│                   │  选项 B: [__________]         │
│                   │  ...                         │
│                   │                               │
│                   │      [ 取消 ]  [ 保存  ]     │
└───────────────────┴──────────────────────────────┘

3. 左侧列表保持不动，右侧显示完整编辑表单
4. 修改完成后点"保存"
5. 左侧列表自动刷新显示更新后的内容
```

优势：
- 可以**一边浏览列表一边编辑**，方便对照
- 编辑区域更大，适合修改选项、上传文件等
- 保存后立即看到更新效果

```

2. 点击该行的 "🗑 删除" 图标
3. 弹出确认对话框:

│                              │
```

```
操作步骤:
1. 勾选要删除的题目（表格左侧复选框 ☐）
   ☑ 101  数学  单选
   ☑ 102  数学  判断
   ☐ 103  英语  阅读
   ☑ 104  物理  多选

4. 确认后批量删除
```

> **重要提示**：删除操作**不可恢复**！除非之前有数据库备份。

```
2. 点击 "👁 预览" 图标
3. 弹出预览窗口:

┌──────────────────────────────────────────────┐

│  📝 题目预览 (学生视角)                       │
```

## 八、批量导入题目

当需要大量添加题目时，逐一手动添加效率太低。批量导入功能支持通过 **JSON 文件**或 **Excel 文件**一次性导入多道题目。

### 8.1 JSON 格式导入

#### JSON 文件格式要求

```json
[
  {
    "subjectId": 1,
    "subcategoryId": 5,
    "type": "single_choice",
    "difficulty": 3,
    "content": "设集合 A={1,2,3}, B={2,3,4}, 则 A∩B=?",
    "options": {
      "A": "{2, 3}",
      "B": "{1, 2, 3, 4}",
      "C": "{1, 4}",
      "D": "{}"
    },
    "answer": "A",
    "explanation": "A∩B 表示取两个集合的交集..."
  },
  {
    "subjectId": 1,
    "subcategoryId": 5,
    "type": "judgment",
    "difficulty": 1,
    "content": "若 a>b，则 a²>b²",
    "options": null,
    "answer": "B",
    "explanation": "反例: a=-1, b=-2 时..."
  }
]
```

#### 字段说明

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| `subjectId` | 是 | Number | 学科 ID（对应 subjects 表） |
| `subcategoryId` | 是 | Number | 子分类 ID |
| `type` | 是 | String | `single_choice` / `multiple_choice` / `judgment` / `reading` |
| `difficulty` | 是 | Number | 1-5 |
| `content` | 是 | String | 题目内容（支持 HTML） |
| `options` | 视题型 | Object/null | 选项对象 `{A:"...", B:"...", ...}`，判断题为 null |
| `answer` | 是 | String | 正确答案（单选/判断为单个字母，多选为多个字母如 "AC"） |
| `explanation` | 否 | String | 答案解析 |

#### 导入步骤

```
1. 准备符合格式的 JSON 文件
2. 点击 "批量导入" → "JSON 导入"
3. 上传 .json 文件
4. 系统解析并校验数据格式
5. 显示预览（有多少道题、哪些有效/无效）
6. 确认后执行导入
7. 显示导入结果（成功 N 道，失败 M 道）
```

### 8.2 Excel 格式导入

#### Excel 文件模板

Excel 导入更直观，适合不熟悉 JSON 的老师使用。

**推荐模板格式（第一行为表头）**：

| subject | subcategory | type | difficulty | content | option_A | option_B | option_C | option_D | answer | explanation |
|---------|-------------|------|------------|---------|----------|----------|----------|----------|---------|------------|
| 数学 | 集合与函数 | single_choice | 3 | 设A={1,2,3}... | {2,3} | {1,2,3,4} | {1,4} | {} | A | 交集定义... |
| 数学 | 集合与函数 | judgment | 1 | 若a>b则a²>b² | | | | | | B | 反例... |
| 英语 | 阅读理解 | reading | 3 | 阅读以下文章... | (见下方) | | | | | (见下方) | ... |

**阅读理解的 Excel 格式**：

阅读理解比较特殊，因为一道题包含多个小题。有两种处理方式：

**方式一：每个小题一行**（同一篇文章重复）

| subject | subcategory | type | difficulty | passage | question | opt_A | opt_B | opt_C | opt_D | answer |
|---------|-------------|------|------------|----------|---------|---------|-------|-------|-------|-------|---------|
| 英语 | 阅读理解 | reading | 3 | 文章全文... | 小题1的问题 | A选项 | B选项 | C选项 | D选项 | A |
| 英语 | 阅读理解 | reading | 3 | 文章全文... | 小题2的问题 | A选项 | B选项 | C选项 | D选项 | B |

**方式二：单独文件**（推荐）

将阅读材料放在一个独立的位置，然后小题引用。

#### 导入步骤

```
1. 按 Excel 模板整理题目数据
2. 点击 "批量导入" → "Excel 导入"
3. 上传 .xlsx 或 .xls 文件
4. 系统自动识别列名并映射字段
5. 映射确认（如果列名不标准，手动指定哪列对应哪个字段）
6. 预览解析后的数据
7. 确认后导入
```

### 8.3 导入注意事项

| 注意事项 | 说明 |
|----------|------|
| **数据量** | 每次建议不超过 500 道，大量题目可分批导入 |
| **字符编码** | 确保 UTF-8 编码，避免中文乱码 |
| **学科/分类 ID** | 确保引用的学科和分类 ID 已存在 |
| **选项格式** | 严格按照 A/B/C/D（或更多）的键名 |
| **预览检查** | 导入前务必预览，检查是否有格式错误 |
| **备份** | 大规模导入前建议先备份数据库 |

## 九、批量操作

### 9.1 批量修改难度

```
场景: 某批题目难度设置不准确，需要统一调整

操作步骤:
1. 筛选出需要修改的题目（用搜索/筛选/手动勾选）
2. 勾选目标题目（复选框 ☑）
3. 点击 "批量操作" → "修改难度"
4. 弹出对话框:
   ┌──────────────────────┐
   │  选择新的难度:           │
   │  ○ ⭐  ○ ⭐⭐  ○ ⭐⭐⭐  │
   │  ● ⭐⭐⭐⭐            │
   │                      │
   │  已选: 15 道题目      │
   │  [ 取消 ]  [ 确认 ]   │
   └──────────────────────┘
5. 确认后所有选中题目的难度统一更新
```

### 9.2 批量移动分类

```
场景: 学科结构调整，需要把一批题移到另一个分类

操作步骤:
1. 勾选目标题目
2. 点击 "批量操作" → "移动分类"
3. 选择目标学科和子分类
4. 确认移动
```

### 9.3 批量修改题型

```
场景: 发现一批题目标注了错误的题型

操作步骤:
1. 勾选目标题目
2. 点击 "批量操作" → "修改题型"
3. 选择正确的题型
4. 确认修改
```

> 注意：**修改题型可能导致选项格式不兼容**（例如从单选改为多选后，原来的单选答案可能无效）。修改后应抽查确认。

## 十一、常见问题

### Q1: 为什么找不到刚添加的题目？

可能原因：
- 筛选条件过滤掉了（检查学科/题型/难度筛选）
- 排序导致不在第一页（翻到最后面看看）
- 刷新页面（缓存问题）

### Q2: 富文本编辑器输入的公式显示异常？

确保使用的是编辑器内置的**公式编辑模式**，不要直接粘贴外部 HTML/LaTeX 代码。

### Q3: 上传图片/音频失败？

- 检查文件大小（最大 10MB）
- 检查文件格式（图片: jpg/png/gif/webp；音频: mp3/wav/ogg）
- 检查网络连接（上传需要调用后端 API）
- 检查后端 `images/` 和 `audio/` 目录是否存在且有写入权限

### Q4: 批量导入报错？

- 检查 JSON 格式是否合法（可以用在线 JSON 校验工具）
- 检查必填字段是否都有值
- 检查学科/分类 ID 是否存在于数据库
- 检查选项的键名是否为 A/B/C/D（大写字母）
- 查看**导入结果**中的错误提示信息

### Q5: 删除的题目能恢复吗？

如果**没有备份**，无法直接恢复。这就是为什么强调重要操作前要先备份的原因。如果有之前的 SQL 备份文件，可以通过 `mysql` 命令恢复单张表的数据。

### Q6: 如何快速复制一道相似的题目？

1. 找到原题目
2. 点击"编辑"
3. 修改需要变动的部分（如改几个数字）
4. 保存为新题目（如果支持"另存为"功能）或记住内容后手动新增

有些系统支持"复制"功能，可以直接基于现有题目创建副本再修改。
