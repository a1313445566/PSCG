# 管理员登录

## 首次使用 - 初始化管理员

**重要提示**：系统没有默认管理员账号，首次使用需要初始化。

### 初始化流程

#### 方法 1：通过管理端页面
1. 访问管理端地址（如 `http://localhost:5173/admin`）
2. 系统会自动跳转到初始化页面
3. 填写管理员信息：
   - **用户名**：设置管理员用户名
   - **密码**：设置管理员密码（至少 6 位）
   - **确认密码**：再次输入密码确认
4. 点击「初始化」按钮
5. 系统提示初始化成功

#### 方法 2：通过 API 接口
```bash
POST /api/admin/init
Content-Type: application/json

{
  "username": "admin",
  "password": "123456"
}
```

### 初始化说明
- 每个系统只能初始化一次
- 初始化后无法更改用户名
- 请妥善保管管理员账号
- 密码使用 bcrypt 加密存储

## 登录流程

### 1. 访问管理端登录页面
- 打开浏览器，输入管理端地址（如 `http://localhost:5173/admin`）
- 系统显示管理员登录页面

### 2. 输入登录信息
- **用户名**：输入管理员用户名
- **密码**：输入管理员密码

### 3. 点击登录
- 点击「登录」按钮
- 系统验证信息，成功后进入管理后台
- 登录状态保存在 sessionStorage 中

## 密码管理

### 修改密码

#### 通过管理端页面
1. 登录管理后台
2. 点击右上角用户头像
3. 选择「修改密码」
4. 输入信息：
   - **原密码**：当前使用的密码
   - **新密码**：新的密码（至少 6 位）
   - **确认新密码**：再次输入新密码
5. 点击「保存」

#### 通过 API 接口
```bash
PUT /api/admin/password
Authorization: Bearer {admin_token}
Content-Type: application/json

{
  "oldPassword": "123456",
  "newPassword": "newpassword123"
}
```

### 忘记密码

**重要提示**：系统不支持在线重置管理员密码。

#### 解决方案

##### 方法 1：数据库重置（推荐）
1. 登录 MySQL 数据库
2. 执行 SQL 语句重置密码：
```sql
-- 查看 admin_credentials 表
SELECT * FROM admin_credentials;

-- 重置密码（需要重新哈希）
UPDATE admin_credentials 
SET password_hash = '新的哈希密码' 
WHERE username = 'admin';
```

##### 方法 2：联系技术支持
- 联系系统开发者或技术支持
- 提供管理员用户名和身份证明
- 由技术人员协助重置密码

##### 方法 3：重新初始化（不推荐）
- 清空 `admin_credentials` 表
- 重新访问管理端初始化
- **注意**：会丢失原管理员数据

## 退出登录

### 方法 1：通过用户菜单
1. 点击页面右上角的用户头像
2. 在下拉菜单中选择「退出登录」
3. 系统清除登录状态
4. 返回登录页面

### 方法 2：清除浏览器数据
- 清除浏览器缓存和 sessionStorage
- 重新打开页面会回到登录状态

---

## 登录状态说明

### Token 信息
- **类型**：JWT Token
- **有效期**：24 小时
- **存储位置**：sessionStorage
- **自动过期**：24 小时后自动失效

### 会话超时
- **无操作超时**：30 分钟无操作自动退出
- **Token 过期**：Token 过期后需要重新登录
- **自动检测**：系统会自动检测并提示

### 多设备登录
- 支持多个设备同时登录
- 没有设备数量限制
- 每个设备独立会话

## 安全说明

### 账号安全
⚠️ **重要提醒**：
1. 管理员账号具有最高权限，请妥善保管
2. 不要将账号密码告诉他人
3. 定期修改密码，提高安全性
4. 不要在公共电脑上登录管理员账号

### 密码强度建议
- 至少 8 位字符
- 包含大小写字母
- 包含数字
- 包含特殊字符
- 避免使用常见密码

### 操作安全
- 重要操作前请确认
- 定期备份数据
- 不要随意删除数据
- 注意操作日志

## 常见问题

### Q1: 忘记管理员密码怎么办？
**A**: 系统不支持在线重置，需要通过数据库修改或联系技术支持。

### Q2: 可以修改管理员用户名吗？
**A**: 不支持修改用户名，只能修改密码。

### Q3: 可以创建多个管理员吗？
**A**: 目前系统只支持一个管理员账号。

### Q4: 管理员和学生可以同一账号吗？
**A**: 不可以，管理员和学生是独立的账号体系。

### Q5: 登录提示"未授权"怎么办？
**A**: 
- 检查用户名密码是否正确
- 确认账号已初始化
- 清除浏览器缓存重试

### Q6: Token 过期了怎么办？
**A**: 重新登录即可获取新的 Token。
