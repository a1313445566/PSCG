> PSCG 智能学习系统 - 管理端 | 版本：v1.0 | 最后更新：2026-04-05

## 一、管理后台入口

### 1.1 访问地址

管理后台与学生端共享同一个系统，通过不同的路径访问：

```
http://localhost:5173/admin          # 本地开发环境
http://你的服务器地址/admin         # 生产环境
```

> **注意**：管理后台路径为 `/admin`，与学生的 `/login` 登录页不同。

### 1.2 访问方式

管理后台**不需要学生登录**即可访问（根据路由配置，`/admin` 和 `/docs` 页面无需登录）。进入后会看到管理员专用的登录界面。

## 五、认证机制说明

### 5.1 与学生端的区别

| 对比项 | 学生端 | 管理端 |
|--------|--------|--------|
| 登录路径 | `/login` | `/admin`（内置） |
| 认证方式 | 学号验证 | 用户名 + 密码 |
| Token 类型 | JWT (student) | JWT (admin) |
| 权限范围 | 仅学习功能 | 全部管理功能 |
| 路由守卫 | 需要登录才能访问 | 无需登录即可进入（内部再验证） |

### 5.2 管理员权限验证

管理后台的 API 接口通过中间件 [`middleware/adminAuth.js`](file:///e:/PSCG/middleware/adminAuth.js) 进行权限验证：

```javascript
// 所有 /api/admin/* 路由都需要管理员 Token
app.use('/api/admin', adminAuth, adminRoutes)
```

未登录或 Token 过期时：
- 前端会拦截 401 响应
- 自动跳转回管理后台登录界面
- 提示"请先登录"

### 5.3 会话管理

- 管理员 Token 同样存储在浏览器 `localStorage` 中
- 过期后需重新登录
- 建议：管理操作完成后主动退出

## 六、常见问题

### Q1: 忘记管理员密码怎么办？

联系技术负责人或查看初始配置。如需重置密码，可通过数据库直接修改（需要数据库访问权限），或运行初始化脚本。

### Q2: 可以多个管理员同时登录吗？

可以。每个管理员有独立的登录会话，互不影响。但建议避免同时修改同一份数据。

### Q3: 管理后台打不开？

排查步骤：
1. 确认后端服务正在运行（`npm run server`）
2. 确认前端开发服务器正在运行（`npm run dev`）
3. 直接访问 `http://localhost:5173/admin`
4. 检查浏览器控制台是否有错误

### Q4: 移动端可以使用管理后台吗？

可以访问，但管理后台主要为**桌面端设计**。部分复杂功能（如题目编辑、数据分析）在小屏幕上体验不佳，建议使用电脑操作。

### Q5: 如何新增管理员账号？

当前版本需要在数据库中手动添加管理员记录，或在代码中进行配置。后续版本可能支持通过界面创建。
