# 管理员操作指南

## 登录管理后台

1. **访问管理后台**
   - 在浏览器地址栏输入 `#/admin`
   - 或使用管理员账号登录后点击「管理后台」

2. **输入管理员账号**
   - 输入管理员用户名和密码
   - 点击「登录」

## 题目管理

### 查看题目列表
1. 点击左侧菜单「题目管理」
2. 查看所有题目列表
3. 支持按学科、子分类、题型筛选
4. 支持关键词搜索

### 添加题目
1. 点击「添加题目」按钮
2. 填写题目信息：
   - **学科**: 选择题目的学科
   - **子分类**: 选择题目的子分类
   - **题目内容**: 输入题目文本
   - **选项**: 输入选项内容（A、B、C、D）
   - **答案**: 输入正确答案
   - **题型**: 选择单选或多选
   - **难度**: 设置题目难度
3. 点击「保存」

### 批量导入题目
1. 点击「批量导入」按钮
2. 选择学科和子分类
3. 粘贴题目文本（按格式）
4. 点击「导入」

**题目格式示例：**
```
1. 题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4

【多选】多选题(ABC)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

### 编辑题目
1. 在题目列表中找到要编辑的题目
2. 点击「编辑」按钮
3. 修改题目信息
4. 点击「保存」

### 删除题目
1. 在题目列表中找到要删除的题目
2. 点击「删除」按钮
3. 确认删除

## 学科管理

### 查看学科
1. 点击左侧菜单「学科管理」
2. 查看所有学科列表
3. 显示每个学科的题目数量

### 添加学科
1. 点击「添加学科」按钮
2. 输入学科名称
3. 选择图标
4. 设置排序
5. 点击「保存」

### 管理子分类
1. 在学科列表中点击「子分类」
2. 查看该学科下的所有子分类
3. 可以添加、编辑、删除子分类

## 用户管理

### 查看用户
1. 点击左侧菜单「用户管理」
2. 查看所有学生列表
3. 支持按年级、班级筛选
4. 支持按学号、姓名搜索

### 添加用户
1. 点击「添加用户」按钮
2. 输入学生信息：
   - 学号
   - 姓名
   - 年级
   - 班级
3. 点击「保存」

### 查看用户详情
1. 点击用户列表中的「详情」
2. 查看该学生的：
   - 基本信息
   - 答题统计
   - 错题记录
   - 学习进度

## 数据分析

### 查看仪表盘
1. 点击左侧菜单「仪表盘」
2. 查看系统概览数据：
   - 总用户数
   - 总题目数
   - 今日答题次数
   - 正确率趋势

### 查看学习统计
1. 点击「学习统计」
2. 查看各学科的答题情况
3. 查看各班级的平均成绩
4. 导出统计报表

## AI 助手

### 配置 AI 模型
1. 点击左侧菜单「AI 模型」
2. 查看已配置的模型列表
3. 点击「添加模型」
4. 配置模型参数：
   - 模型名称
   - 提供商
   - API 地址
   - API Key
   - 模型标识
5. 点击「保存」

### 使用 AI 聊天
1. 点击左侧菜单「AI 助手」
2. 进入聊天界面
3. 选择要使用的模型
4. 输入问题开始对话
5. 可以查看历史会话

### 查看 Token 使用
1. 点击「Token 统计」
2. 查看 Token 使用情况
3. 查看成本统计

## 系统设置

### 基础设置
1. 点击左侧菜单「系统设置」
2. 配置系统参数：
   - 系统名称
   - 每页显示数量
   - 答题时间限制
   - 其他参数

### 数据备份
1. 点击「数据备份」
2. 可以手动备份数据库
3. 查看备份历史
4. 恢复数据
