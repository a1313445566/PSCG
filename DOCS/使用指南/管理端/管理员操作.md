## 二、仪表盘概览

### 2.1 入口

登录管理后台后**默认页面**即为仪表盘，也可点击侧边栏 **"🏠 仪表盘"** 进入。

```

```

### 2.3 仪表盘内容

仪表盘提供系统的**全局数据概览**，通常包括：

| 展示项 | 说明 |
|--------|------|
| **学生总数** | 注册的学生人数 |
| **题目总数** | 题库中的题目总量 |

### 3.3 功能概述

题目管理是管理后台**最核心的功能**之一，提供完整的题库生命周期管理：

| 功能 | 说明 |
|------|------|
| **浏览题目** | 以表格形式展示所有题目，支持分页、搜索、筛选 |
| **新增题目** | 通过表单添加单个新题目（支持富文本、图片、音频） |
| **编辑题目** | 修改已有题目的内容和选项 |
| **删除题目** | 删除单个或批量删除题目 |
| **预览题目** | 查看题目的实际渲染效果（学生视角） |
| **批量导入** | 通过 JSON 或 Excel 文件批量导入题目 |
| **批量操作** | 批量修改难度、移动分类、删除等 |

### 3.4 操作流程

#### 浏览和搜索题目

```
1. 进入题目管理页面
2. 使用顶部工具栏筛选条件：
   - 关键词搜索（题目标题/内容）
   - 学科筛选
   - 题型筛选（单选/多选/判断/阅读理解）
   - 难度筛选
   - 子分类筛选
3. 表格显示匹配结果
4. 点击分页浏览更多
```

#### 新增题目

```
1. 点击工具栏的 "+ 新增题目" 按钮
2. 弹出题目编辑表单，填写：
   - 学科和子分类（必选）
   - 题型（必选）
   - 难度等级（1-5星）
   - 题目标题（支持富文本编辑器 Quill）
   - 选项内容（A/B/C/D，根据题型不同）
   - 正确答案
   - 解析（可选）
   - 上传配图（可选，支持 jpg/png）
   - 上传音频（可选，支持 mp3/wav 等）
3. 点击"保存"
```

#### 编辑题目

```
方式一：行内编辑
  1. 在表格中找到目标题目
  2. 点击该行的"编辑"按钮
  3. 直接在表格中修改（适合简单修改）

方式二：分屏编辑
  1. 点击某行的"编辑"或双击题目
  2. 右侧/弹窗打开完整编辑面板
  3. 左侧显示题目列表，右侧显示编辑表单
  4. 适合复杂编辑（修改选项、上传文件等）
```

#### 删除题目

```
单个删除:
  1. 找到目标题目
  2. 点击"删除"按钮
  3. 确认弹窗 → 确认删除

批量删除:
  1. 勾选多个题目（表格左侧复选框）
  2. 点击工具栏"批量删除"
  3. 确认弹窗 → 确认批量删除
```

#### 预览题目

```
1. 找到目标题目
2. 点击"预览"按钮
3. 弹出预览窗口，以学生视角展示：
   - 题目内容的实际渲染效果
   - 选项的实际样式
   - 图片/音频的实际展示
4. 确认无误后关闭预览
```

#### 批量导入

```
JSON 格式导入:
  1. 点击"批量导入" → 选择 "JSON 导入"
  2. 上传符合格式的 .json 文件
  3. 系统解析并预览导入的数据
  4. 确认无误后执行导入

Excel 格式导入:
  1. 点击"批量导入" → 选择 "Excel 导入"
  2. 上传 .xlsx/.xls 文件
  3. 系统自动映射列（题目标题→title, 选项A→optionA...）
  4. 预览并确认导入
```

#### 批量操作

```
批量修改难度:
  1. 勾选多个题目
  2. 点击"批量操作" → "修改难度"
  3. 选择新的难度值
  4. 确认修改

批量移动分类:
  1. 勸选多个题目
  2. 点击"批量操作" → "移动分类"
  3. 选择目标学科和子分类
  4. 确认移动
```

## 五、数据分析

### 5.3 分析维度

| 分析项 | 说明 |

| 能力 | 说明 |

| `classAnalysisTool` | 班级分析 |
| `classComparisonTool` | 班级对比 |

| `wrongQuestionsTool` | 错题查询 |
| `learningPathTool` | 学习路径推荐 |
| `knowledgePointMasteryTool` | 知识点掌握度 |
| `weakPointsTool` | 薄弱点分析 |
| `systemOverviewTool` | 系统概况 |
| `timeTrendTool` | 时间趋势 |
| `personalizedRecommendationTool` | 个性化推荐 |
| `classFullAnalysisTool` | 班级全面分析 |
| `getStudentAnswerDetailTool` | 学生答题详情 |
| `getQuestionAnswerAnalysisTool` | 题目答案分析 |

### 6.5 操作指引

```
1. 进入 AI 聊天界面
2. 在底部输入框输入问题，例如：
   - "查询数学学科的平均正确率"
   - "分析高一(1)班的学习情况"
   - "找出错误率最高的 5 道题"
3. AI 自动调用合适的工具获取数据
4. AI 整合数据后以自然语言回复
5. 可继续追问深入分析
```

## 十一、操作注意事项

### 11.1 权限提醒

- 管理员拥有**最高权限**，请谨慎操作
- 删除操作**不可恢复**（除非有备份）
- 批量操作前务必**确认选中项**

### 11.2 数据安全

- 定期使用**备份恢复**功能备份数据库
- 重要操作前建议**手动备份**
- 不要在生产环境直接做实验性操作

### 11.3 性能注意

- 题目导入时避免一次性导入过多（建议每次 < 500 条）
- 数据分析查询大范围数据时可能较慢，请耐心等待
- 执行数据清理时建议在低峰期进行
