# 批量添加题目指南

## 功能概述

批量添加题目功能允许管理员一次性导入多个题目到指定的学科题库中，支持单选题和多选题，可以自动解析题目内容和选项。

## 批量添加题目格式

### 支持的格式

#### 格式1: 带编号的题目
```
1. 题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4

2. 题目内容(B)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

#### 格式2: 无编号的题目
```
题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

#### 格式3: 多选题
```
【多选】题目内容(ABC)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

#### 格式4: 带解析的题目
```
题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
解析: 这里是解析内容
```

### 格式规则

#### 1. 题目行格式
- 支持编号格式: `1. 题目内容(A)` 或 `1、题目内容(A)`
- 支持无编号格式: `题目内容(A)`
- 答案括号格式: `(A)`、`(a)`、`（A）`、`（a）`
- 支持答案括号后有标点: `(A).`、`(A)`

#### 2. 选项行格式
- `A. 选项内容` 或 `A、选项内容` 或 `A．选项内容`
- 支持行首有空格
- 选项字母不区分大小写

#### 3. 解析行格式
- `解析: 内容` 或 `解析：内容`
- 支持行首有空格

#### 4. 多选题识别
- 通过【多选】标记自动识别
- 通过答案长度自动识别(如(ABC)为多选)

## 批量添加逻辑

### 1. 解析流程

解析过程按照以下步骤进行:

1. 将输入文本按行分割
2. 逐行匹配题目模式:
   - 带编号题目: `/^(\d+[.、]\s*)(.+?)([\(（]\s*[A-Da-d]+\s*[\)）])(.*)$/`
   - 无编号题目: `/^(.+?)([\(（]\s*[A-Da-d]+\s*[\)）])(.*)$/`
3. 提取题目内容和答案
4. 自动识别题目类型(单选/多选)
5. 解析选项行
6. 解析解析行(可选)

### 2. 题目类型识别

系统会自动识别题目类型:

```javascript
// 自动识别题目类型
let questionType = 'single';
// 检查是否有【多选】标记
if (questionText.includes('【多选】') || answer.length > 1) {
  questionType = 'multiple';
}
```

识别规则:
- 如果题目文本包含【多选】标记，则识别为多选题
- 如果答案长度大于1(如ABC、ABCD)，则识别为多选题
- 否则识别为单选题

### 3. 数据处理

解析后的题目会被转换为以下格式:

```javascript
{
  subjectId: batchSubjectId.value,
  subcategoryId: batchSubcategoryId.value,
  type: question.type,
  content: question.content,
  options: question.options,
  answer: question.answer,
  explanation: question.explanation
}
```

### 4. 使用步骤

1. 选择学科
2. 选择学科题库
3. 粘贴题目文本
4. 点击"解析题目"按钮预览
5. 确认无误后点击"批量添加"

## 注意事项

### 格式要求
1. 每个题目之间需要空行分隔
2. 选项必须以字母开头(A、B、C、D)
3. 答案括号中的字母会被移除,只保留空括号
4. 支持中英文标点符号
5. 解析是可选的,可以不提供

### 数据验证
- 必须选择学科和学科题库
- 题目文本不能为空
- 每个题目必须有至少2个选项
- 答案必须在选项范围内

### 常见错误
1. **格式错误**: 题目行缺少答案括号
2. **选项错误**: 选项不以字母开头
3. **答案错误**: 答案不在选项范围内
4. **分隔错误**: 题目之间没有空行

## 技术实现

### 组件位置
- 文件路径: `src/components/admin/question-management/BatchAddQuestion.vue`

### 核心功能
1. 文本解析: 使用正则表达式匹配题目模式
2. 类型识别: 自动识别单选题和多选题
3. 预览功能: 实时预览解析结果
4. 数据验证: 验证题目格式和完整性

### 正则表达式说明

#### 带编号题目匹配
```javascript
/^(\d+[.、]\s*)(.+?)([\(（]\s*[A-Da-d]+\s*[\)）])(.*)$/
```
- `^(\d+[.、]\s*)`: 匹配题目编号(如1.、1、)
- `(.+?)`: 匹配题目内容
- `([\(（]\s*[A-Da-d]+\s*[\)）])`: 匹配答案括号
- `(.*)$`: 匹配括号后的内容

#### 无编号题目匹配
```javascript
/^(.+?)([\(（]\s*[A-Da-d]+\s*[\)）])(.*)$/
```
- `^(.+?)`: 匹配题目内容
- `([\(（]\s*[A-Da-d]+\s*[\)）])`: 匹配答案括号
- `(.*)$`: 匹配括号后的内容

#### 选项匹配
```javascript
/^\s*([A-Za-z][\.\、．]?\s*)(.*)$/
```
- `^\s*`: 匹配行首空格
- `([A-Za-z][\.\、．]?\s*)`: 匹配选项标签
- `(.*)$`: 匹配选项内容
