# 🏢 PSCG 后台管理系统使用指南

## 📋 概述

PSCG后台管理系统是为管理员提供的完整管理平台，包含题库管理、用户管理、数据分析、系统设置等核心功能。

## 🚀 快速开始

### 1. 访问后台
1. 打开浏览器，访问：`http://您的域名/admin`
2. 使用管理员账号登录
3. 进入后台管理界面

### 2. 导航结构
后台管理系统采用标签页式导航，包含以下主要功能模块：
- **基础设置** - 系统基础配置
- **学科管理** - 学科和题库管理
- **题目管理** - 题目CRUD操作
- **用户管理** - 用户信息管理
- **排行榜管理** - 排行榜数据查看
- **数据分析** - 学习数据分析
- **数据管理** - 数据备份恢复

## 📚 功能模块详解

### 1. 基础设置

#### 界面名称设置
- **功能**：设置系统显示名称
- **操作步骤**：
  1. 在基础设置标签页找到"界面名称设置"
  2. 输入新的系统名称
  3. 点击"保存设置"按钮
- **效果**：系统所有页面将显示新名称

#### 答题设置
- **功能**：配置答题相关参数
- **可配置项**：
  - 选项随机排序：开启/关闭
  - 固定题目数量：开启/关闭
  - 最小题目数量：数值设置
  - 最大题目数量：数值设置
- **操作步骤**：
  1. 修改相应配置项
  2. 点击"保存设置"按钮

### 2. 学科管理

#### 添加学科
1. 点击"添加学科"按钮
2. 填写学科信息：
   - 学科名称
   - 选择图标
   - 排序序号
3. 点击"确认添加"

#### 编辑学科
1. 在学科列表中找到目标学科
2. 点击"编辑"按钮
3. 修改学科信息
4. 点击"保存"

#### 删除学科
1. 在学科列表中找到目标学科
2. 点击"删除"按钮
3. 确认删除操作

> **注意**：删除学科将同时删除该学科下的所有题库和题目。

#### 题库管理
1. 点击学科右侧的"管理题库"按钮
2. 在弹出窗口中：
   - 添加新题库
   - 编辑现有题库
   - 删除题库
   - 调整题库顺序

### 3. 题目管理

#### 题目列表
- **查看题目**：按学科、题库筛选
- **搜索题目**：支持关键词搜索
- **排序功能**：按创建时间、难度排序
- **分页浏览**：支持分页查看

#### 添加题目
1. 点击"添加题目"按钮
2. 选择所属学科和题库
3. 填写题目信息：
   - 题目内容（支持富文本）
   - 选项（A、B、C、D）
   - 正确答案
   - 难度等级（1-5星）
   - 解析说明
4. 上传附件（图片/音频）
5. 点击"保存"

#### 批量添加题目
1. 点击"批量添加"按钮
2. 按照指定格式输入题目：
   ```
   1. 题目内容(A)
   A. 选项1
   B. 选项2
   C. 选项3
   D. 选项4
   ```
3. 选择目标题库
4. 点击"导入"按钮

#### 编辑题目
1. 在题目列表中点击"编辑"按钮
2. 修改题目信息
3. 更新附件（可选）
4. 点击"保存"

#### 删除题目
1. 在题目列表中点击"删除"按钮
2. 确认删除操作

#### 题目筛选和搜索
- **按学科筛选**：选择特定学科
- **按题库筛选**：选择特定题库
- **按难度筛选**：选择难度等级
- **关键词搜索**：搜索题目内容
- **组合筛选**：支持多重条件组合

### 4. 用户管理

#### 用户列表
- 查看所有注册用户
- 查看用户详细信息
- 筛选用户（按年级、班级）

#### 用户详情
1. 点击用户列表中的"查看详情"按钮
2. 查看用户信息：
   - 基本信息
   - 学习统计
   - 答题记录
   - 成绩趋势

#### 用户统计
- 总用户数统计
- 活跃用户统计
- 学习进度分析
- 成绩分布统计

### 5. 排行榜管理

#### 全局排行榜
- 查看所有用户排名
- 按学科筛选排名
- 查看个人详细排名

#### 近期记录
- 查看最近的答题记录
- 分析答题趋势
- 识别热门题库

#### 用户统计
- 个人成绩统计
- 学科学习情况
- 进步趋势分析

### 6. 数据分析

#### 学习数据分析
- 整体学习情况统计
- 学科难度分析
- 用户活跃度分析
- 答题正确率分析

#### 错误率分析
- 高错误率题目识别
- 错误原因分析
- 改进建议

#### 趋势分析
- 学习趋势图表
- 成绩变化分析
- 预测分析

### 7. 数据管理

#### 数据备份
1. 点击"数据备份"按钮
2. 选择备份类型：
   - 完整备份
   - 增量备份
3. 点击"开始备份"
4. 下载备份文件

#### 数据恢复
1. 点击"数据恢复"按钮
2. 上传备份文件
3. 确认恢复操作
4. 等待恢复完成

#### 数据清理
1. 点击"数据清理"按钮
2. 选择清理类型：
   - 清理答题记录
   - 清理临时文件
   - 清理缓存数据
3. 确认清理操作

#### 数据导出
- 导出题目数据（Excel格式）
- 导出用户数据
- 导出学习统计

## 🔧 高级功能

### 1. 难度自动调整

#### 功能说明
系统根据用户答题情况自动调整题目难度：
- 错误率高的题目降低难度
- 正确率高的题目提高难度
- 基于时间加权调整

#### 操作步骤
1. 进入"题目管理"
2. 选择"难度调整"标签
3. 查看难度分布
4. 手动调整（可选）

### 2. 错误率分析

#### 功能说明
识别错误率较高的题目，帮助改进教学质量。

#### 使用步骤
1. 进入"数据分析"
2. 选择"错误率分析"
3. 查看错误率排名
4. 分析错误原因

### 3. 批量操作

#### 批量导入
支持从文本文件批量导入题目。

#### 批量导出
支持将题目导出为Excel文件。

#### 批量修改
支持批量修改题目属性。

## 🚨 常见问题

### 1. 题目导入失败
- **问题**：批量导入题目时格式错误
- **解决**：检查题目格式是否符合要求

### 2. 图片上传失败
- **问题**：图片上传后无法显示
- **解决**：检查图片格式和大小限制

### 3. 数据备份失败
- **问题**：数据备份过程中断
- **解决**：检查磁盘空间和权限

### 4. 用户统计不准确
- **问题**：用户统计数据与实际不符
- **解决**：检查数据同步设置
