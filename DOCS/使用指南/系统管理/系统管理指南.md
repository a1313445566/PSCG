# 🛠️ PSCG 系统管理指南

## 📋 概述

本指南面向系统管理员，详细介绍了PSCG系统的部署、配置、维护、监控和故障排除等管理操作。

## 🏗️ 系统架构

### 1. 技术栈概览
```
前端技术栈：
├── Vue 3 + Composition API
├── Pinia 状态管理
├── Vue Router 路由
├── Element Plus UI组件库
├── ECharts 数据可视化
└── Vite 构建工具

后端技术栈：
├── Node.js 16+
├── Express.js 4.x
├── MySQL 8.0 数据库
├── Multer 文件上传
├── PM2 进程管理
└── 多种中间件和工具
```

### 2. 系统架构图
```
客户端 → Nginx反向代理 → Node.js服务器 → MySQL数据库
        ↓               ↓               ↓
      静态资源        业务逻辑        数据存储
        ↓               ↓               ↓
      CDN缓存        缓存层          备份系统
```

## 🚀 部署指南

### 1. 环境要求

#### 硬件要求
- **CPU**：2核以上
- **内存**：4GB以上
- **存储**：50GB以上可用空间
- **带宽**：10Mbps以上

#### 软件要求
- **操作系统**：Ubuntu 20.04+/CentOS 7+
- **Node.js**：16.x 或更高版本
- **MySQL**：8.0 或更高版本
- **Nginx**：1.18+（可选）
- **PM2**：5.0+（进程管理）

#### 网络要求
- **端口开放**：3001（API）、80/443（Web）
- **域名**：已备案的域名（生产环境）
- **SSL证书**：HTTPS支持

### 2. 安装步骤

#### 2.1 基础环境准备
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装Node.js
curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt install -y nodejs

# 安装MySQL
sudo apt install -y mysql-server
sudo mysql_secure_installation

# 安装Nginx（可选）
sudo apt install -y nginx

# 安装PM2
sudo npm install -g pm2
```

#### 2.2 项目部署
```bash
# 克隆项目
git clone <repository-url>
cd PSCG

# 安装依赖
npm install

# 创建必要目录
mkdir -p audio images uploads backups

# 设置目录权限
chmod 755 audio images uploads backups
chown www:www audio images uploads backups

# 构建前端
npm run build
```

#### 2.3 数据库配置
```bash
# 登录MySQL
sudo mysql -u root -p

# 创建数据库
CREATE DATABASE quiz_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 创建用户
CREATE USER 'quiz_user'@'localhost' IDENTIFIED BY 'strong_password';
GRANT ALL PRIVILEGES ON quiz_db.* TO 'quiz_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

#### 2.4 配置文件
```bash
# 创建生产环境配置
cp .env.production .env

# 编辑配置文件
nano .env
# 设置以下内容：
# VITE_API_URL=http://您的域名:3001/api
# 其他环境变量...
```

### 3. 启动服务

#### 3.1 使用PM2启动
```bash
# 启动后端服务
pm2 start server.cjs --name "pscg-api"

# 设置开机自启
pm2 startup
pm2 save

# 查看服务状态
pm2 status
pm2 logs pscg-api
```

#### 3.2 Nginx配置（反向代理）
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    # 重定向到HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name your-domain.com;
    
    # SSL证书配置
    ssl_certificate /path/to/certificate.crt;
    ssl_certificate_key /path/to/private.key;
    
    # 静态资源服务
    location / {
        root /www/wwwroot/PSCG/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # API代理
    location /api {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

### 3. 安全配置

#### 文件上传限制
编辑 `server.cjs`：
```javascript
const upload = multer({
    storage: storage,
    limits: {
        fileSize: 10 * 1024 * 1024,  // 10MB限制
        files: 1                      // 每次1个文件
    },
    fileFilter: (req, file, cb) => {
        // 文件类型白名单
        const allowedTypes = ['image/jpeg', 'image/png', 'image/gif', 'image/webp', 'audio/mpeg', 'audio/wav'];
        if (allowedTypes.includes(file.mimetype)) {
            cb(null, true);
        } else {
            cb(new Error('不支持的文件类型'));
        }
    }
});
```

#### CORS配置
```javascript
app.use(cors({

    credentials: true,

```

### 2. 数据备份

#### 自动备份脚本
创建 `/etc/cron.daily/pscg-backup`：
```bash
#!/bin/bash
BACKUP_DIR="/www/wwwroot/PSCG/backups"
DATE=$(date +%Y-%m-%d-%H-%M-%S)
BACKUP_FILE="$BACKUP_DIR/backup-$DATE-full.db"

# 备份数据库
mysqldump -u quiz_user -p'your_password' quiz_db > $BACKUP_FILE

# 压缩备份文件
gzip $BACKUP_FILE

# 保留最近7天备份
find $BACKUP_DIR -name "*.db.gz" -mtime +7 -delete

# 发送备份通知
echo "PSCG数据库备份完成: $BACKUP_FILE.gz" | mail -s "PSCG备份通知" admin@example.com
```

#### 手动备份
```bash
# 完整备份
cd /www/wwwroot/PSCG
node server.cjs --backup full

# 增量备份
node server.cjs --backup incremental

# 恢复备份
node server.cjs --restore backup-2025-03-23-10-00-00-full.db
```
