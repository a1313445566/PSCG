# 学生登录与答题

## 登录系统

1. **访问登录页面**
   - 打开浏览器，访问系统地址
   - 默认显示登录页面

2. **输入登录信息**
   - **学号**: 输入您的学号
   - **年级**: 选择您所在的年级
   - **班级**: 选择您所在的班级

3. **点击登录**
   - 点击「登录」按钮
   - 系统验证信息后进入首页

## 选择学科和分类

1. **浏览学科**
   - 首页显示所有学科卡片
   - 点击学科卡片进入子分类页面

2. **选择子分类**
   - 查看该学科下的所有子分类
   - 显示每个子分类的题目数量
   - 点击子分类开始答题

## 开始答题

1. **答题界面**
   - 显示当前题目内容和选项
   - 选项会自动乱序
   - 显示答题进度

2. **选择答案**
   - 点击选项选择答案
   - 单选题只能选择一个答案
   - 多选题可以选择多个答案

3. **切换题目**
   - 点击「下一题」继续答题
   - 系统会自动保存答题记录

4. **提交答案**
   - 完成所有题目后点击「提交」
   - 系统会显示答题结果

## 错题巩固模式

1. **进入错题巩固**
   - 在子分类页面选择「错题巩固」
   - 系统会加载您答错的题目

2. **错题规则**
   - 只显示累计答对次数少于 3 次的错题
   - 按最近错题时间排序
   - 答对 3 次后自动从错题库移除

## 查看学习数据

1. **学习进度**
   - 点击「学习进度」查看统计数据
   - 显示各学科的答题情况

2. **错题本**
   - 点击「错题本」查看所有错题
   - 可以重新练习错题

3. **排行榜**
   - 点击「排行榜」查看排名
   - 显示班级和全局排名

4. **学习报告**
   - 点击「学习报告」查看详细分析
   - 包含正确率、答题趋势等数据
