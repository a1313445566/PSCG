# 登录与答题完整流程

> PSCG 智能学习系统 - 学生端 | 版本：v1.0 | 最后更新：2026-04-05

---

## 一、完整流程概览

```
登录 → 首页(选学科) → 子分类(选知识点) → 答题 → 提交 → 查看结果 → 错题复习
```

下面按顺序详细介绍每一步的操作。

---

## 二、第一步：登录系统

打开浏览器访问系统地址（如 `http://localhost:5173`），进入登录页面。

### 界面位置

```
页面路径: /login
组件: LoginForm.vue (src/components/common/LoginForm.vue)
视图: LoginView.vue (src/views/LoginView.vue)
```

### 操作步骤

1. 在 **学号** 输入框输入你的学号
2. 点击 **"登录"** 按钮
3. 登录成功后自动跳转到**首页**

> 新用户请先参考 [注册登录.md](./注册登录.md) 完成注册。

### 各区域功能说明

#### 顶部导航栏

| 元素 | 位置 | 功能 |
|------|------|------|
| "< 返回" | 左上角 | 放弃本次答题，返回子分类页（有确认弹窗） |
| 学科 > 子分类 | 中间偏左 | 显示当前位置（面包屑导航） |
| "第 X / Y 题" | 中间 | 当前进度和总题数 |
| 计时器 ⏱ | 右侧 | 本次答题已用时间（分:秒格式） |

#### 题目区域 (QuestionCard)

- 显示**题号**、**题型标签**（单选/多选/判断/阅读理解）、**难度星级**
- 显示**题目内容**（支持富文本：加粗、公式、图片等）
- 如有**配图**：显示在题目下方
- 如有**音频**：显示播放器控件

#### 选项区域 (OptionList)

- **单选题**：点击即选中（单选），再次点击其他选项切换
- **多选题**：点击勾选/取消（可多选），需点击确定提交答案
- **判断题**：两个按钮 ["对"] ["错"]
- **阅读理解**：先显示文章，下方跟随小题

#### 底部操作区

| 按钮 | 功能 |
|------|------|
| 上一题 | 切换到前一题（第一题时不可用） |
| 标记本题 | 标记为"待复查"，答题卡中特殊标识 |
| 下一题 | 切换到后一题（最后一题时可提交） |

#### 答题卡

- 显示所有题目的作答状态
- 图例：✅ 已作答 / ◯ 未作答 / ⭐ 已标记 / ❌ 当前题
- 点击编号可快速跳转到对应题目
- **"提交答卷"** 按钮：完成答题后提交

## 六、第五步：逐题作答详解

### 单选题操作

```
初始状态:
  A. 选项A内容          ← 普通
  B. 选项B内容
  C. 选项C内容
  D. 选项D内容

点击选项 B 后:
  A. 选项A内容
  B. ✔ 选项B内容        ← 高亮选中（蓝色边框+背景）
  C. 选项C内容
  D. 选项D内容

→ 自动保存答案为 B，可随时更改
```

### 多选题操作

```
初始状态:
  ☐ A. 选项A内容         ← 空白方框
  ☐ B. 选项B内容
  ☐ C. 选项C内容
  ☐ D. 选项D内容

点击 A 和 C:
  ☑ A. 选项A内容         ← 选中（带勾方框）
  ☐ B. 选项B内容
  ☑ C. 选项C内容         ← 选中
  ☐ D. 选项D内容

→ 多选题需手动确认，建议选完后点击"确定"或切题时自动确认
```

### 判断题操作

```
  ┌─────────┐    ┌─────────┐
  │   对 ✓  │    │   错   │    ← 两个大按钮
  │ (正确)  │    │ (错误)  │
  └─────────┘    └─────────┘

点击"对" → 选中高亮 → 答案记录为 A（对）
点击"错" → 选中高亮 → 答案记录为 B（错）
```

## 七、第六步：提交答卷

### 7.1 提交方式

有两种方式可以提交答卷：

**方式一：通过答题卡提交**
1. 点击底部的 **"提交答卷"** 按钮
2. 弹出确认对话框
3. 确认后提交

**方式二：最后一题点"下一题"**
1. 答完最后一题后点击 **"下一题"**
2. 自动弹出提交确认

### 7.2 提交确认弹窗

```

3. 显示得分、正确率、答案详细对比

### 界面位置

```
页面路径: /error-book
视图: ErrorBookView.vue (src/views/ErrorBookView.vue)
```

### 错题本功能

- **错题列表**：按时间/学科筛选查看错题
- **重新作答**：针对错题再次练习
- **移出错题**：掌握后可以从错题本移除
- **查看解析**：查看正确答案和解题思路

## 十、完整流程速查

```
1. 打开浏览器 → 输入系统地址
2. 输入学号 → 点击"登录"
3. 首页 → 点击学科卡片（如"数学"）
4. 子分类页 → 点击知识点（如"集合与函数"）
5. 答题页 → 逐题作答 → 选答案 → 下一题
