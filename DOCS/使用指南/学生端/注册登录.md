> PSCG 智能学习系统 - 学生端 | 版本：v1.0 | 最后更新：2026-04-05

## 一、系统访问

### 1.1 访问地址

在浏览器中输入系统地址：

```
http://localhost:5173          # 本地开发环境
http://你的服务器地址           # 生产环境（如 http://your-domain.com）
```

首次访问会自动跳转到**登录页面** (`/login`)。

### 1.2 浏览器兼容性

| 浏览器 | 最低版本 | 推荐版本 |
|--------|----------|----------|
| Chrome | 90+ | 最新版 |
| Firefox | 88+ | 最新版 |
| Edge | 90+ | 最新版 |
| Safari | 14+ | 最新版 |

> **注意**：推荐使用 Chrome 或 Edge 以获得最佳体验。

## 二、注册新账号

### 2.1 注册入口

如果还没有账号，在登录页面点击 **"注册"** 按钮（或 **"立即注册"** 链接）切换到注册表单。

### 2.2 注册信息填写

注册页面需要填写以下信息：

| 字段 | 说明 | 示例 | 必填 |
|------|------|------|------|
| **学号** | 你的唯一标识，用于登录 | `20240101` | 是 |
| **姓名** | 你的真实姓名 | `张三` | 是 |
| **年级** | 所在年级 | `高一` / `高二` / `高三` 等 | 是 |
| **班级** | 所在班级 | `1班` / `2班` 等 | 是 |

### 2.4 注册规则

- **学号**：必须唯一，不能与已有用户重复
- **姓名**：2-20 个字符，支持中文
- **年级/班级**：从下拉列表中选择，选项由管理员配置
- 注册成功后**自动登录**并跳转到首页

### 3.1 登录步骤

1. 打开系统地址，进入登录页面
2. 在 **学号** 输入框中输入你的学号
3. 点击 **"登录"** 按钮

### 3.3 登录验证过程

```
输入学号 → 点击登录
    │
    ▼
系统验证学号是否存在
    │
    ├── 存在 → 生成 JWT Token → 存储到浏览器 → 跳转首页 ✓
    │
    └── 不存在 → 显示错误提示："该学号未注册，请先注册" ✗
```

### 3.4 Token 存储

登录成功后，系统会自动存储以下信息到浏览器：

| 存储项 | 存储位置 | 用途 | 有效期 |
|--------|----------|------|--------|
| `studentId` | localStorage | 标识用户身份 | 手动清除前有效 |
| Token (JWT) | localStorage | 身份验证凭证 | **24 小时** |
| `tokenExpiresAt` | localStorage | Token 过期时间戳 | - |
| `lastActivity` | sessionStorage | 最后活动时间 | **30 分钟**无活动过期 |

## 四、会话管理

### 4.1 Token 有效期

- **Token 有效期**：24 小时（自登录时起计算）
- **活动超时**：30 分钟内没有任何操作，会话将自动失效

### 4.2 会话超时机制

系统采用双重超时机制：

```
每次操作（点击、导航等）
    │
    ▼
更新 lastActivity 时间戳（sessionStorage）
    │
    ├── 下次操作时检查：
    │   ├── Token 是否过期？（对比 tokenExpiresAt 与当前时间）
    │   │   └── 已过期 → 清除所有数据 → 跳转登录页
    │   │
    │   └── 会话是否超时？（当前时间 - lastActivity > 30分钟）
    │       └── 已超时 → 清除所有数据 → 跳转登录页
```

### 4.3 超时后重新登录

当会话超时或 Token 过期时：

1. 页面自动跳转到 `/login`
2. 显示提示信息（如需要可添加友好提示）
3. 使用学号重新登录即可继续使用

## 五、退出登录

### 5.1 主动退出

在系统中找到 **退出登录** 入口：

- **位置**：通常在页面右上角的个人头像/菜单区域
- **操作**：点击 **"退出登录"** 或 **"登出"** 按钮

### 5.2 退出后的处理

退出登录时，系统会执行以下操作：

```javascript
// 清除所有本地存储的数据
localStorage.clear()
sessionStorage.clear()

// 跳转到登录页
window.location.href = '/login'
```

退出后将无法访问需要登录的页面（首页、答题、排行榜等），会被自动重定向回登录页。

### 5.3 安全建议

- 在**公共电脑**上使用完毕后，务必点击**退出登录**
- **不要**直接关闭浏览器（虽然 localStorage 不会因此清除，但这是好习惯）
- 如果忘记退出，30 分钟无活动后会自动过期

## 六、常见问题

### Q1: 忘记学号怎么办？

**解决方法**：
- 联系你的老师或管理员查询你的学号
- 或让管理员在后台查看用户列表

### Q2: 提示"该学号已存在"怎么办？

**原因**：这个学号已经被其他同学注册过了

**解决方法**：
- 确认你输入的是否是自己的正确学号
- 如果确认学号正确但显示已存在，可能是之前已经注册过，直接用该学号登录即可
- 如果确实没有注册过但提示已存在，请联系管理员处理

### Q3: 选择不到自己的年级/班级？

**原因**：管理员还未在后台配置对应的年级或班级选项

**解决方法**：
- 联系管理员在 **基础设置 → 年级班级管理** 中添加你的年级和班级
- 配置完成后刷新注册页面即可看到新选项

### Q4: 可以重复注册吗？

**不可以**。每个学号只能注册一个账号。如果需要修改个人信息（如姓名、班级），请联系管理员。

### Q5: 注册后可以修改密码吗？

当前版本的登录机制基于**学号验证**（非密码模式），无需记忆密码。只要知道学号就可以登录。

### Q6: 手机/平板可以使用吗？

**可以**。系统采用响应式设计，支持手机、平板和桌面设备访问。推荐横屏使用以获得更好的答题体验。

### Q7: 登录后一直跳回登录页面？

可能的原因及排查：

1. **Token 未正确存储**：检查浏览器是否禁用了 localStorage
2. **后端服务未启动**：确认 API 服务正常运行（联系管理员）
3. **网络问题**：检查网络连接是否正常
4. **浏览器缓存问题**：尝试清除缓存或使用隐私模式访问
