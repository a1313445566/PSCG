# 👨🎓 PSCG 学生使用指南

## 📋 概述

PSCG学生端是为学生提供的在线学习平台，支持智能答题、错题巩固、学习分析、排行榜查看等功能，帮助学生高效学习和提高成绩。

## 🚀 快速开始

### 1. 访问系统
1. 打开浏览器，访问：`http://您的域名`
2. 如果是首次使用，请先注册账号
3. 登录后进入学习主页

### 2. 首次使用设置
1. **完善个人信息**：设置年级、班级
2. **选择学习目标**：设定学习计划
3. **了解功能**：浏览系统功能介绍

## 📚 核心功能详解

### 1. 智能答题系统

#### 开始答题
1. 在主页点击"开始答题"按钮
2. 选择学科和题库
3. 设置答题参数：
   - 题目数量（10-50题）
   - 难度级别（自动/指定）
   - 是否随机排序选项
4. 点击"开始答题"

#### 答题界面
- **题目展示**：清晰显示题目和选项
- **多媒体支持**：支持图片和音频题目
- **进度显示**：显示答题进度
- **时间提示**：显示已用时间

#### 答题操作
- **选择答案**：点击选项选择答案
- **下一题**：完成当前题后自动/手动进入下一题
- **跳过题目**：暂时跳过，稍后回答
- **结束答题**：提前结束答题

#### 答题反馈
- **即时反馈**：选择答案后立即显示正确/错误
- **答案解析**：显示题目解析
- **难度调整**：根据答题情况动态调整后续题目难度

### 2. 答题结果分析

#### 结果总览
- **得分统计**：正确率、得分
- **用时统计**：总用时、平均每题用时
- **难度分析**：题目难度分布

#### 错题回顾
1. 答题结束后自动进入错题回顾
2. 查看所有答错的题目
3. 查看正确答案和解析
4. 重新学习错题

#### 学习建议
- 基于答题情况提供个性化建议
- 推荐需要加强的领域
- 建议学习计划

### 3. 错题巩固功能

#### 错题本
- **自动收集**：系统自动记录所有错题
- **分类整理**：按学科、题库、错误类型分类
- **智能复习**：根据遗忘曲线推荐复习

#### 错题练习
1. 进入"错题巩固"页面
2. 选择要复习的错题
3. 开始专项练习
4. 掌握后标记为"已掌握"

#### 错题分析
- **错误类型分析**：分析常见错误类型
- **薄弱环节识别**：识别需要加强的知识点
- **进步跟踪**：跟踪错题掌握进度

### 4. 学习数据分析

#### 个人统计
- **总体统计**：总答题数、正确率、学习时长
- **学科统计**：各学科学习情况
- **时间统计**：每日/每周/每月学习情况

#### 进度跟踪
- **学习进度**：显示学习计划完成情况
- **成绩趋势**：显示成绩变化趋势
- **能力提升**：显示能力提升情况

#### 成绩分析
- **成绩分布**：各分数段分布情况
- **进步情况**：与历史成绩对比
- **能力评估**：综合能力评估

### 5. 排行榜系统

#### 查看排行榜
1. 点击导航栏"排行榜"
2. 选择排行榜类型：
   - 全局排行榜
   - 学科排行榜
   - 班级排行榜

#### 排行榜功能
- **排名查看**：查看自己的排名
- **用户对比**：与优秀学生对比
- **进步激励**：查看进步情况

#### 成就系统
- **学习成就**：完成学习任务获得成就
- **答题成就**：答题表现获得成就
- **进步成就**：进步明显获得成就

### 6. 学习计划管理

#### 制定计划
1. 进入"学习计划"页面
2. 设置学习目标：
   - 每日学习时间
   - 每周学习目标
   - 月度学习计划
3. 保存计划

#### 计划跟踪
- **进度显示**：显示计划完成情况
- **提醒功能**：学习提醒
- **调整功能**：根据实际情况调整计划

#### 成就奖励
- **完成奖励**：完成计划获得奖励
- **连续奖励**：连续学习获得奖励
- **进步奖励**：显著进步获得奖励

## 🎮 学习技巧

### 1. 高效答题技巧

#### 时间管理
- **合理分配时间**：根据题目难度分配时间
- **跳过难题**：先跳过难题，最后回来解答
- **检查答案**：留出时间检查答案

#### 答题策略
- **排除法**：先排除明显错误的选项
- **关键词法**：注意题目中的关键词
- **整体把握**：理解题目整体意思

### 2. 错题学习方法

#### 错题分析步骤
1. **记录错题**：及时记录所有错题
2. **分析原因**：分析错误原因
3. **分类整理**：按类型分类整理
4. **定期复习**：按照遗忘曲线复习

#### 错题掌握标准
- **完全理解**：理解题目和解析
- **能够讲解**：能够给别人讲解
- **举一反三**：能够解决类似问题

### 3. 学习计划制定

#### 制定原则
- **具体可行**：目标要具体可衡量
- **循序渐进**：从易到难，循序渐进
- **灵活调整**：根据情况灵活调整

#### 执行技巧
- **每日坚持**：养成每日学习习惯
- **及时反馈**：及时查看学习效果
- **适当奖励**：完成目标后适当奖励

## 🚨 常见问题

### 1. 答题问题
- **问题**：答题过程中页面卡顿
- **解决**：刷新页面或重新开始答题

### 2. 登录问题
- **问题**：无法登录系统
- **解决**：检查账号密码或联系管理员

### 3. 数据问题
- **问题**：学习数据不准确
- **解决**：刷新页面或清除缓存

### 4. 网络问题
- **问题**：网络连接不稳定
- **解决**：检查网络连接或稍后重试
