# 查看成绩与排名

## 查看个人成绩

### 1. 登录系统
- 输入学号、姓名、年级、班级登录系统
- 进入系统首页

### 2. 进入学习报告页面
- 在导航栏中点击「学习报告」
- 进入个人学习报告页面

### 3. 查看成绩概览
系统显示您的学习数据概览：
- **总答题数**：累计答题数量
- **总正确率**：所有题目的正确率
- **学习时长**：总学习用时
- **积分**：当前积分

### 4. 查看详细数据

#### 学科统计
- 各学科的答题情况
- 每个学科的正确率
- 学习进度展示

#### 答题记录
- 历史答题记录列表
- 每次答题的时间、题数、正确率
- 点击记录可以查看详情

## 查看排行榜

### 1. 进入排行榜页面
- 在导航栏中点击「排行榜」
- 进入排行榜页面

### 2. 班级排行榜
- 显示您所在班级的排名
- 列表包含：排名、姓名、学号、积分
- 您的排名会高亮显示

### 3. 全局排行榜
- 显示所有学生的排名
- 列表包含：排名、姓名、学号、班级、积分
- 可以查看其他同学的积分情况

### 4. 排行榜说明
- **排名依据**：按积分从高到低排序
- **积分规则**：
  - 普通答题：答对一题得 1 分，答错一题扣 1 分，全对积分翻倍
  - 错题巩固：每道题累计答对 3 次 +1 分
- **更新频率**：实时或准实时更新

## 积分规则说明

### 普通答题积分
```
基础分 = 答对题数 - 答错题数
全对奖励 = 基础分 × 2（全部答对时）
```

**示例**：
- 10 题答对 8 题，答错 2 题：积分 = 8 - 2 = 6 分
- 10 题全部答对：积分 = (10 - 0) × 2 = 20 分

### 错题巩固积分
- 每道错题累计答对 3 次：+1 分
- 再次答错：重置计数，不得分

### 积分用途
- **排行榜排名**：积分决定排名
- **学习激励**：鼓励多答题、答对题
- **学习成果**：反映学习投入和效果

## 排名规则

### 排名依据
- **主要依据**：积分从高到低排序
- **积分相同**：按答题总数排序
- **仍然相同**：按正确率排序

### 排名分类
1. **班级排名**：仅显示同班同学
2. **全局排名**：显示所有学生

### 排名更新
- 答题提交后积分立即更新
- 排行榜准实时刷新
- 可能存在短暂延迟

## 注意事项

⚠️ **重要提醒**：

1. **数据准确性**：排名数据可能有一定延迟，属正常现象
2. **隐私保护**：排行榜只显示姓名、学号、班级和积分
3. **公平竞争**：系统会检测异常答题行为
4. **理性看待**：排名只是参考，重要的是学习进步
5. **持续努力**：多答题、答对题可以提高排名

## 常见问题

### Q1: 为什么我的排名没有更新？
**A**: 可能原因：
- 数据更新有延迟，稍等片刻
- 网络异常，刷新页面
- 积分未达到更新条件

### Q2: 积分相同为什么排名不同？
**A**: 积分相同时，按答题总数排序，答题多的排名靠前。

### Q3: 可以看到别人的详细信息吗？
**A**: 只能看到排行榜上的基本信息（姓名、学号、班级、积分）。

### Q4: 如何提高排名？
**A**: 
- 多答题，增加积分
- 提高答题正确率
- 参与错题巩固

### Q5: 排行榜多久更新一次？
**A**: 准实时更新，答题后积分会立即或稍后更新。

### Q6: 可以退出排行榜吗？
**A**: 不可以，排行榜是系统功能，所有学生都会参与排名。
