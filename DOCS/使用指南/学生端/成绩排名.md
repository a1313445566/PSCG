## 一、功能总览

学生端提供以下学习和成绩相关功能：

| 功能 | 路径 | 主要用途 |
|------|------|----------|
| 排行榜 | `/leaderboard` | 查看自己在全班的排名情况 |

| 信息 | 说明 |
|------|------|
| 总体进度条 | 所有学科的综合完成比例 |

- 建议均衡发展各学科，避免偏科

### 4.4 错题本功能

#### 查看错题列表

默认按**时间倒序**排列（最新的在前），每条错题显示：

- 错题序号和基本信息（学科、题型、知识点）
- 做题日期
- 题目摘要
- 你的答案 vs 正确答案

#### 筛选错题

| 筛选条件 | 选项 |
|----------|------|
| 学科 | 全部 / 数学 / 语文 / 英语 / ... |
| 题型 | 全部 / 单选 / 多选 / 判断 / 阅读 |
| 关键词 | 搜索题目内容中的关键词 |

#### 重新作答

点击 **"重新作答"** 按钮：

```
点击"重新作答"
    │
    ▼
进入该错题的单独练习模式
    │
    ├── 只显示这一道错题
    ├── 作答后立即显示对错和解析
    └── 如果答对了，可以选择从错题本移除
```

#### 查看解析

点击 **"查看解析"** 展开详细的解题思路和答案解析。

#### 移出错题

当你已经**完全掌握**某道错题后：

```
点击"移除"按钮
    │
    ▼
弹出确认: "确定要从错题本中移除此题吗？"
    │
    ├── 确认 → 从错题本删除（可恢复）
    └── 取消 → 保留
```

> 建议：重新作答且连续答对 2 次后再移除，确保真正掌握。

### 6.4 历史记录信息

每次答题都会生成一条历史记录，包含：

| 信息 | 说明 |
|------|------|
| 时间 | 答题完成的精确时间 |
| 学科 > 知识点 | 答题所属的学科和子分类 |
| 得分 | 本次答题的百分制得分 |
| 正确率 | 答对题目占比 |
| 用时 | 从开始到提交的总时长 |
| 题数/正确/错误 | 题目统计数据 |

### 6.5 查看详情

点击 **"查看详情"** 可以回看某次答题的完整情况：

- 每道题的作答情况
- 答案对比
- 当时的时间消耗
- 可选择"重新作答"再练一次

### 6.6 时间线视图

答题历史按**时间倒序**排列（最新在前），按日期分组显示，形成清晰的学习时间线。你可以：

- 回顾自己每天的学习量
- 找出学习效率高的日子并总结经验
- 发现遗漏的学习日（没有记录的日子）
