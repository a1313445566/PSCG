# AI 智能对话系统 - 使用指南

## 📋 系统概述

PSCG 智能题库系统现已集成 AI 对话功能，提供：
- 🤖 **智能对话助手**：基于 LangChain 的多轮对话能力
- 📊 **数据分析工具**：8个专业数据分析工具
- 🔧 **模型管理**：支持多种大模型配置和切换
- 💾 **智能缓存**：提升响应速度，降低成本

---

## 🚀 快速开始

### 1. 访问 AI 助手

登录管理后台后，在左侧菜单中找到：
- **AI 助手**：智能对话界面
- **模型管理**：配置和管理 AI 模型

### 2. 配置模型

首次使用前，需要在「模型管理」中添加 AI 模型：

#### 步骤：
1. 点击「添加模型」按钮
2. 填写模型信息：
   - **模型名称**：例如 `GPT-4`
   - **模型类型**：`openai` / `deepseek` / `anthropic` / `zhipu`
   - **API Key**：您的 API 密钥（加密存储）
   - **模型标识**：例如 `gpt-4-turbo-preview`
   - **温度参数**：建议 `0.7`（0-2之间）
   - **最大Token**：建议 `4000`

3. 点击「测试连接」验证配置
4. 保存后设置为默认模型

## 🔧 高级功能

### 1. 会话管理

- **新建会话**：点击左上角「+ 新建会话」
- **会话历史**：自动保存所有对话记录
- **删除会话**：右键会话列表可删除

### 2. 模型切换

在聊天界面顶部可快速切换已配置的模型：
- 不同模型有不同的能力和价格
- 建议日常使用性价比高的模型
- 复杂分析可切换到高级模型

### 3. 上下文压缩

系统自动进行上下文压缩：
- 保留关键信息
- 减少Token消耗
- 降低API调用成本

## 📊 性能优化

### 缓存机制

系统采用两级缓存策略：

#### L1 缓存（内存）
- 热点数据快速响应
- 缓存时间：5分钟
- 命中率监控：实时显示

#### L2 缓存（数据库）
- 持久化缓存数据
- 缓存时间：24小时
- 自动清理过期数据

### Token 统计

每次对话都会记录：
- 输入Token数
- 输出Token数
- 总消耗成本

可在模型管理中查看详细统计。

## 🔒 安全机制

### 1. API Key 加密存储

- 使用 AES-256-GCM 加密
- 密钥存储在环境变量中
- 传输过程加密

### 2. 访问权限控制

- 仅管理员可访问 AI 功能
- JWT Token 验证
- CSRF 防护

### 3. 速率限制

- 防止滥用 API
- 每分钟请求限制
- 超限自动拦截

## 🛠️ 故障排查

### 常见问题

#### 1. 模型连接失败
**原因**：API Key 无效或网络问题  
**解决**：
- 检查 API Key 是否正确
- 确认网络连接正常
- 查看后端日志

#### 2. 对话响应慢
**原因**：模型处理时间长或缓存未命中  
**解决**：
- 检查模型配置
- 查看缓存命中率
- 考虑切换更快的模型

#### 3. Token 消耗过快
**原因**：对话上下文过长  
**解决**：
- 开启上下文压缩
- 清除历史会话
- 调整 max_tokens 参数

## 📝 API 接口文档

### 聊天接口

#### POST `/api/chat/sessions/:sessionId/messages`

发送消息并获取 SSE 流式响应

**请求体**：
```json
{
  "content": "查询三年级学生的平均正确率"
}
```

**响应**（SSE 流）：
```
data: {"type":"content","content":"正在查询..."}
data: {"type":"tool_call","tool":"studentStats","args":{...}}
data: {"type":"content","content":"三年级学生的平均正确率为 75.3%"}
data: [DONE]
```

### 模型管理接口

#### GET `/api/chat/models`
获取所有模型列表

#### POST `/api/chat/models`
添加新模型

#### PUT `/api/chat/models/:id`
更新模型配置

#### DELETE `/api/chat/models/:id`
删除模型

#### POST `/api/chat/models/:id/test`
测试模型连接

## 🎯 最佳实践

### 1. 提问技巧

✅ **推荐**：
- "查询三年级一班数学学科的平均正确率"
- "分析最近一周错题率最高的 5 道题目"
- "对比张三和李四的学习进度"
- "何骏铭的学习情况如何？有什么改进建议？"

AI 助手现在会提供：
- 📊 **深度分析**：不只是数字罗列，会解读数据背后的含义
- 🔍 **洞察与建议**：发现问题原因，提供具体改进建议
- 📈 **对比维度**：班级平均、年级排名、历史对比
- 🎓 **教育专业性**：基于教育学理论的专业分析

❌ **不推荐**：
- "查询数据"（过于模糊）
- "所有学生的信息"（范围太大）

### 2. 成本控制

- 使用缓存：避免重复查询
- 合理设置 max_tokens
- 选择性价比高的模型
- 定期清理无用会话

### 3. 性能优化

- 开启压缩：减少上下文长度
- 使用索引：数据库查询优化
- 监控缓存：查看命中率统计
