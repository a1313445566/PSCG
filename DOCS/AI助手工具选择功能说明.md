# AI助手工具选择功能说明

## 功能概述

新增了AI助手快捷提问和工具选择功能，让老师可以：
1. **快速选择学生/班级**：直接选择查询对象
2. **选择专用工具**：根据需求选择合适的数据分析工具
3. **快捷提问**：基于选中的工具和实体，自动生成推荐问题

## 使用方法

### 1. 打开AI助手
- 访问管理后台 → 点击左侧菜单"AI助手"
- 或通过快捷入口：学生查询、班级查询、预警查询

### 2. 显示工具栏
在聊天输入框上方，点击"显示工具栏"按钮

### 3. 选择学生/班级
在工具栏中：
- 选择类型：学生 或 班级
- 选择年级：1-6年级
- 选择班级：1-10班
- 选择学生（如果查询学生）

### 4. 选择工具
工具分为7大类：

#### 学生分析类
- **学生学习统计**：查询学生答题次数、正确率、积分
- **学生预警**：识别学习异常学生（连续错误、正确率下降、未登录）

#### 班级分析类
- **班级学情分析**：查询班级学生人数、平均正确率、答题总数
- **班级深度分析**：包含进步退步学生的详细分析
- **班级对比分析**：对比不同班级的学习情况

#### 知识点分析类
- **薄弱知识点**：查询学生或班级的薄弱知识点
- **知识点掌握度**：查询各知识点的掌握程度
- **错题分析**：分析错题情况

#### 学习轨迹类
- **时间趋势分析**：学习数据随时间变化趋势
- **学习轨迹追踪**：完整学习轨迹（知识点演进、能力变化）
- **学科进度分析**：各学科的进步情况

#### 个性化推荐类
- **个性化推荐**：生成个性化学习建议和题目推荐

#### 题目查询类
- **题目详情**：查询题目详细信息
- **错题本查询**：查询学生错题本

#### 系统概览类
- **系统概览**：系统整体统计数据

### 5. 使用快捷提问
选择工具后，会自动显示推荐问题：
- 问题模板中的占位符会自动替换为选中的学生/班级名称
- 点击问题即可快速发送

### 6. 发送消息
- 输入框支持普通文本输入
- 选择工具后，消息会自动标注使用的工具
- 按 `Ctrl + Enter` 发送消息

## API接口

### 获取工具元数据
```
GET /api/tools/metadata
```

**响应示例**：
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "tools": [
      {
        "name": "query_student_stats",
        "displayName": "学生学习统计",
        "category": "student",
        "icon": "User",
        "description": "查询学生答题次数、正确率、积分",
        "quickQuestions": [
          "查看{studentName}的学习统计数据",
          "{grade}年级{studentName}最近的学习情况如何"
        ]
      }
    ],
    "categories": [
      { "id": "student", "name": "学生分析", "icon": "User" }
    ]
  }
}
```

### 获取指定分类的工具
```
GET /api/tools/category/:category
```

### 获取单个工具详情
```
GET /api/tools/detail/:name
```

## 使用场景示例

### 场景1：查询学生小明（3年级1班）的学习情况
1. 点击"显示工具栏"
2. 选择"学生" → "3年级" → "1班" → "小明"
3. 选择工具"学生学习统计"
4. 点击快捷问题："查看小明的学习统计数据"
5. AI自动调用工具并返回结果

### 场景2：查询3年级1班的预警学生
1. 点击"显示工具栏"
2. 选择"班级" → "3年级" → "1班"
3. 选择工具"学生预警"
4. 点击快捷问题："3年级1班有哪些学生需要预警"
5. AI返回预警学生列表（连续错误、正确率下降、未登录）

### 场景3：对比3年级各班学习情况
1. 点击"显示工具栏"
2. 选择"班级" → "3年级"
3. 选择工具"班级对比分析"
4. 点击快捷问题："对比3年级各班的学习表现"
5. AI返回各班对比数据

## 技术特点

1. **智能提示**：根据选择的工具和实体，自动生成推荐问题
2. **参数自动填充**：问题模板自动替换为实际学生/班级名称
3. **工具分类**：按功能分类，方便快速查找
4. **响应式设计**：适配不同屏幕尺寸
5. **缓存优化**：工具元数据缓存，减少请求次数

## 注意事项

1. 必须先创建或选择对话，才能使用工具栏
2. 学生列表需要根据年级和班级动态加载
3. 快捷问题数量限制为3-5个，避免界面过于拥挤
4. 工具选择后，消息会自动标注使用的工具名称
5. 所有组件遵循项目规范，使用 `<script setup>` 语法
