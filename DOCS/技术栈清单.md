## 📦 数据库

### 类型与配置
- **类型**: MySQL 8.0+
- **依赖**: `mysql2` ^3.20.0
- **实例位置**: `services/database.js`
- **连接方式**: 连接池 (mysql.createPool)

### 语法规范
```sql
-- ✅ MySQL 语法示例
CREATE TABLE IF NOT EXISTS table_name (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_name (name)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='表注释';

-- ❌ 禁止使用 SQLite 语法
-- INTEGER PRIMARY KEY AUTOINCREMENT  -- 错误！
```

### 数据库操作方法
```javascript
const db = require('./services/database');

// 查询多条记录
const rows = await db.query('SELECT * FROM users LIMIT ?', [10]);

// 查询单条记录
const row = await db.get('SELECT * FROM users WHERE id = ?', [1]);

// 执行更新/插入/删除
await db.run('INSERT INTO users SET ?', { name: '张三', age: 20 });

// 关闭连接
await db.close();
```

## 🎣 Composables (Hook)

### useLoading
**文件**: `src/composables/useLoading.js`

```javascript
import { useLoading } from '@/composables/useLoading'
import { onUnmounted } from 'vue'

const { showLoading, hideLoading, withLoading, cleanup } = useLoading()

// 方式1：手动控制
showLoading('加载中...')
await fetchData()
hideLoading()

// 方式2：自动包装（推荐）
await withLoading(async () => {
  await fetchData()
}, '加载中...')

// 组件卸载时清理
onUnmounted(cleanup)
```

### usePagination
**文件**: `src/composables/usePagination.js`

**⚠️ 重要**: `total` 作为参数传入，不是返回值！

```javascript
import { usePagination } from '@/composables/usePagination'
import { ref } from 'vue'

// ✅ 正确的使用方式
const total = ref(0)  // 先创建 total ref
const pagination = usePagination(20, total)  // 传入 total ref

// 使用分页参数
const params = pagination.getServerParams()  // { page: 1, limit: 20 }

// 更新总数
total.value = result.total  // 直接修改 total ref

// 事件处理
const handleSizeChange = (size) => {
  pagination.handleSizeChange(size)
  loadData()
}

const handleCurrentChange = (page) => {
  pagination.handleCurrentChange(page)
  loadData()
}

// 重置分页
pagination.reset()
```

```javascript
// IP 验证
const isValidIP = (ip) => {
  const ipv4Regex = /^(\d{1,3}\.){3}\d{1,3}$/
  const ipv6Regex = /^([0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}$/
  return ipv4Regex.test(ip) || ipv6Regex.test(ip)
}

// 必填验证
if (!name || name.trim() === '') {
  return res.status(400).json({ error: '姓名不能为空' })
}

// 类型验证
if (typeof age !== 'number' || age < 0) {
  return res.status(400).json({ error: '年龄必须是正整数' })
}
```

## 🚫 禁止事项

### 数据库相关
- ❌ 禁止使用 SQLite 语法（`INTEGER PRIMARY KEY AUTOINCREMENT`）
- ❌ 禁止在代码中拼接 SQL 字符串（必须使用参数化查询）
- ❌ 禁止创建没有索引的大表

### 前端相关
- ❌ 禁止使用原生 `fetch`（必须使用 `api` 封装）
- ❌ 禁止直接使用 `ElMessage`（必须使用 `message` 封装）
- ❌ 禁止忘记导入图标（必须从 `@element-plus/icons-vue` 导入）
- ❌ 禁止错误使用 `usePagination`（`total` 是参数，不是返回值）
- ❌ 禁止在生产环境输出调试日志（仅保留错误日志）

### 后端相关
- ❌ 禁止绕过权限验证（所有管理后台 API 必须使用 `adminAuth`）
- ❌ 禁止忽略错误处理（所有异步操作必须 `try-catch`）
- ❌ 禁止返回敏感信息（如密码、Token 等）
