# 批量添加题目示例

## 批量添加题目功能介绍

批量添加题目是 PSCG 智能题库系统的一个重要功能，允许管理员通过文本文件一次性导入多个题目，大大提高了题目管理的效率。

## 支持的题目格式

系统支持以下格式的题目：

### 单选题格式
```
1. 题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

### 多选题格式
```
【多选】题目内容(ABC)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
```

### 带图片的题目
```
2. 题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
[图片]images/question1.jpg
```

### 带音频的题目
```
3. 题目内容(A)
A. 选项1
B. 选项2
C. 选项3
D. 选项4
[音频]audio/question1.mp3
```

## 导入步骤

1. **准备题目文件**
   - 创建一个文本文件（.txt）
   - 按照上述格式编写题目
   - 确保答案格式正确（单选为单个字母，多选为多个字母）

2. **登录管理后台**
   - 访问 `http://localhost:5173/admin`
   - 输入管理员用户名和密码

3. **进入题目管理**
   - 点击左侧导航栏的「题目管理」
   - 点击「批量导入」按钮

4. **上传文件**
   - 选择要导入的题目文件
   - 选择题目所属的学科
   - 点击「导入」按钮

5. **查看导入结果**
   - 系统会显示导入结果
   - 包括成功导入的题目数量和失败的题目
   - 失败的题目会显示失败原因

## 注意事项

1. **格式要求**
   - 题目编号必须以数字开头，后跟点号
   - 选项必须以大写字母开头，后跟点号
   - 答案必须放在题目内容后的括号中
   - 多选题必须在题目前添加「【多选】」标记

2. **文件编码**
   - 建议使用 UTF-8 编码
   - 避免使用特殊字符

3. **图片和音频**
   - 图片和音频文件必须先上传到对应的目录
   - 路径必须是相对路径，如 `images/question1.jpg`

4. **批量导入限制**
   - 单次导入的题目数量建议不超过 1000 条
   - 导入过程中请耐心等待，不要刷新页面

5. **错误处理**
   - 导入失败的题目会显示具体错误原因
   - 请根据错误信息修改题目格式后重新导入

## 常见错误及解决方案

### 1. 答案格式错误
- **错误信息**: 答案格式错误
- **解决方案**: 确保答案放在题目内容后的括号中，如 `题目内容(A)`

### 2. 选项格式错误
- **错误信息**: 选项格式错误
- **解决方案**: 确保选项以大写字母开头，后跟点号，如 `A. 选项1`

### 3. 题目编号重复
- **错误信息**: 题目编号重复
- **解决方案**: 确保每个题目的编号唯一

### 4. 学科不存在
- **错误信息**: 学科不存在
- **解决方案**: 先创建学科，然后再导入题目

### 5. 文件过大
- **错误信息**: 文件过大
- **解决方案**: 分割成多个文件，分批导入

## 最佳实践

1. **分类管理**
   - 按照学科和难度分类组织题目
   - 使用注释（# 开头）标记分类

2. **批量编辑**
   - 使用文本编辑器批量编辑题目
   - 利用查找替换功能快速修改

3. **质量控制**
   - 导入前仔细检查题目格式
   - 导入后抽样检查题目内容

4. **版本控制**
   - 保留原始题目文件
   - 定期备份题目数据
