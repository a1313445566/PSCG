# AI 数据增强功能使用指南

## 概述

AI 数据增强功能通过人工智能技术，为题库系统提供智能化的分析和建议能力，帮助教师更好地管理题库，帮助学生更高效地学习。

---

## 一、功能列表

### 1. 题目语义分析
- 自动提取题目关键词和标签
- 识别知识点和考点
- 分析难度因素
- 查找相似题目
- 评估题目质量

### 2. 用户学习风格分析
- 分析答题习惯（速度、稳定性、修改次数）
- 识别错误模式（概念错误、计算错误、粗心等）
- 生成个性化学习建议
- 追踪学习进度趋势

### 3. 智能推荐
- 根据学习风格推荐题目
- 根据错误模式推荐复习内容
- 根据难度偏好调整题目推送

## 二、管理员使用指南

### 2.1 配置 AI 功能

1. **访问配置页面**
   - 登录后台管理系统
   - 进入"基础设置" → "AI 配置"

2. **配置参数说明**

   | 参数 | 说明 | 建议值 |
   |------|------|--------|
   | API Key | 豆包 API 密钥 | 从控制台获取 |
   | API 地址 | API 接口地址 | 保持默认 |
   | 模型名称 | 使用的模型 | 保持默认 |
   | 启用状态 | 是否启用 AI 功能 | 开启 |
   | 缓存开关 | 是否启用结果缓存 | 开启 |
   | 超时时间 | API 请求超时时间（秒） | 120 |
   
   **高级配置**：
   
   | 参数 | 说明 | 建议值 |
   |------|------|--------|
   | 最大并发数 | 同时处理的任务数 | 5 |
   | 重试次数 | 失败后的重试次数 | 3 |
   | 重试延迟 | 重试间隔时间（毫秒） | 1000 |
   | 速率限制等待 | 遇到限制时的等待时间（毫秒） | 2000 |
   | 缓存有效期 | 缓存结果的有效时间（秒） | 3600 |

3. **保存配置**
   - 填写完参数后点击"保存配置"
   - 系统会自动验证配置有效性

---

### 2.2 查看题目语义分析

1. **访问路径**
   - 后台管理 → 语义分析

2. **功能说明**
   - 查看已分析的题目列表
   - 搜索和筛选题目
   - 查看详细分析结果

3. **分析结果解读**

   **关键词**: 题目涉及的核心概念
   
   **标签**: AI 自动生成的分类标签
   - 题型标签：单选题、多选题、判断题等
   - 难度标签：简单、中等、困难等
   - 考点标签：计算、理解、应用等
   
   **知识点**: 题目考查的具体知识点
   
   **难度因素**: 影响题目难度的因素分析
   - 计算量：大、中、小
   - 概念复杂度：高、中、低
   - 解题步骤：多、中、少
   
   **质量评分**: 题目内容的综合质量评分（0-100）

---

### 2.3 查看用户学习风格

1. **访问路径**
   - 后台管理 → 学习风格

2. **功能说明**
   - 查看所有用户的学习风格
   - 搜索和筛选用户
   - 查看详细分析报告

3. **学习风格标签解读**

   | 标签 | 特征 | 建议 |
   |------|------|------|
   | 谨慎型 | 答题时间长，修改次数多 | 保持细致，适当提速 |
   | 思考型 | 答题时间适中，正确率高 | 保持状态，持续进步 |
   | 快速型 | 答题时间短，修改少 | 注意审题，避免粗心 |
   | 稳定型 | 答题时间波动小 | 保持节奏，稳步提升 |
   | 进取型 | 偏好高难度题目 | 注意基础，全面发展 |
   | 保守型 | 偏好低难度题目 | 适当挑战，突破自我 |

4. **错误模式分析**

   | 错误类型 | 说明 | 改进建议 |
   |---------|------|---------|
   | 概念错误 | 对知识点理解有误 | 回顾教材，理解概念 |
   | 计算错误 | 运算过程出错 | 多做计算练习，检查过程 |
   | 粗心错误 | 审题不清或笔误 | 认真审题，仔细检查 |
   | 理解错误 | 题目理解偏差 | 多读题，抓住关键信息 |

## 三、学生使用指南

### 3.1 自动行为追踪

系统会自动记录你的答题行为：
- 答题时间
- 答案修改次数
- 犹豫时间
- 跳题情况

**隐私保护**: 所有数据仅用于学习分析，不会泄露给第三方。

### 3.2 查看学习建议

1. **查看路径**
   - 个人中心 → 学习报告
   - 答题结果页面

2. **建议内容**
   - 学习风格分析
   - 薄弱知识点提示
   - 改进建议
   - 推荐练习题目

## 四、教师使用指南

### 4.1 批量分析题目

1. **操作步骤**
   - 进入"题目管理"
   - 选择需要分析的题目
   - 点击"批量操作" → "语义分析"
   - 等待分析完成

2. **注意事项**
   - 单次最多分析50道题目
   - 分析时间约3-5秒/题
   - 已分析的题目会自动跳过

### 4.2 查看班级学习情况

1. **访问路径**
   - 数据分析 → 学习风格分析
   - 选择班级和时间段

2. **分析报告**
   - 班级整体学习风格分布
   - 学生个体学习风格
   - 常见错误模式统计
   - 教学建议

## 六、常见问题

### Q1: AI 分析结果不准确怎么办？

**A**: AI 分析基于模型训练，可能存在误差。建议：
- 检查题目内容是否完整清晰
- 手动调整不准确的结果
- 反馈问题给管理员

### Q2: 学习风格会一直变化吗？

**A**: 学习风格会随着答题数据的积累而动态更新。通常需要至少20次答题才能形成初步的风格分析。

### Q3: 如何提高分析准确性？

**A**: 
- 确保答题行为真实（不要随意作答）
- 保持正常的答题节奏
- 积累足够的答题数据

### Q4: 数据安全如何保障？

**A**:
- 所有数据加密存储
- 仅用于学习分析目的
- 严格遵守隐私保护规定
- 支持数据导出和删除
