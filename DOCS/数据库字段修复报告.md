## 修复方案

### 方案1：修改代码适配数据库（推荐）

修改所有服务文件，使字段名与 database.js 中的定义一致。

**优点**：
- 不影响现有数据
- 改动范围可控
- 向后兼容

**缺点**：
- 需要修改多个文件

### 方案2：修改数据库适配代码

修改 database.js 中的表结构，使其与代码中的字段名一致。

**优点**：
- 代码改动少

**缺点**：
- 可能影响现有数据
- 需要数据库迁移
- 风险较高

---

## 建议

**推荐方案1**：修改代码适配数据库，确保所有字段名与 CREATE TABLE 定义一致。
