#### 核心功能

```javascript
import { usePagination } from '@/composables/usePagination'

// 前端分页
const allUsers = ref([...])  // 100 条数据
const { currentPage, pageSize, paginate } = usePagination(20)
const displayUsers = computed(() => paginate(allUsers.value))

// 服务端分页
const total = ref(0)
const { currentPage, pageSize, getServerParams } = usePagination(20, total)
const params = getServerParams()  // { page: 1, limit: 20 }
```

#### 响应式数据

| 属性 | 类型 | 说明 |
|------|------|------|
| `currentPage` | Ref\<number\> | 当前页码（从 1 开始） |
| `pageSize` | Ref\<number\> | 每页条数 |
| `totalPages` | ComputedRef\<number\> | 总页数（服务端分页） |

#### 核心方法

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `paginate(data)` | Array | Array | 前端分页：对数据进行切片 |
| `getServerParams()` | - | { page, limit } | 服务端分页：生成分页参数 |
| `handleSizeChange(size)` | number | - | 每页条数变化处理 |
| `handleCurrentChange(page)` | number | - | 页码变化处理 |
| `jumpToPage(page)` | number | boolean | 跳转到指定页 |
| `reset()` | - | - | 重置分页状态 |
| `getPaginationInfo()` | - | Object | 获取分页信息 |
