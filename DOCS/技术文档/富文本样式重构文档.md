# 富文本样式重构文档

## 概述

将项目中分散在多个组件的富文本样式代码提取为公共样式文件，统一管理富文本内容的显示样式。

## 改进内容

### 1. 创建公共样式文件

**文件位置**: `src/styles/rich-text.css`

**主要功能**:
- 提供统一的富文本内容样式类 `.rich-text-content`
- 支持多种尺寸变体（small, medium, large, xlarge）
- 支持紧凑型布局 `.rich-text-compact`
- 支持可点击图片 `.rich-text-clickable`

### 2. 样式特性

#### 基础样式
- ✅ 段落间距控制
- ✅ 文本格式（加粗、斜体、下划线、删除线）
- ✅ 图片自适应（最大宽度 100%）
- ✅ 列表样式（有序、无序）
- ✅ 引用块样式
- ✅ 代码块样式
- ✅ 上下标支持
- ✅ 标题样式（H1-H6）
- ✅ 表格样式
- ✅ 链接样式
- ✅ 分隔线样式

#### 尺寸变体
- **size-small**: 图片最大高度 80px（用于表格、列表项）
- **size-medium**: 图片最大高度 150px（用于选项、预览）
- **size-large**: 图片最大高度 300px（用于题目内容）
- **size-xlarge**: 图片最大高度 400px（用于详情对话框）

#### 特殊变体
- **rich-text-compact**: 紧凑型布局，减少间距
- **rich-text-clickable**: 可点击图片，支持图片预览

### 3. 更新的组件

| 组件 | 修改内容 |
|------|----------|
| **QuestionCard.vue** | 题目内容使用 `size-large`，选项使用 `size-medium`，添加 `rich-text-clickable` |
| **QuestionDetailDialog.vue** | 题目内容使用 `size-xlarge`，选项使用 `size-medium` |
| **LeaderboardView.vue** | 题目内容使用 `size-medium`，选项使用 `size-medium` |
| **ErrorAnalysis.vue** | 表格内容使用 `rich-text-compact size-small`，预览使用 `size-large` |
| **QuestionList.vue** | 内容文本和预览框使用公共样式（待完成） |

## 使用方法

### 基础用法

```vue
<template>
  <!-- 题目内容 -->
  <div class="rich-text-content size-large" v-html="question.content"></div>
  
  <!-- 答案选项 -->
  <div class="rich-text-content size-medium" v-html="option"></div>
</template>

<style scoped>
@import '@/styles/rich-text.css';
</style>
```

### 可点击图片（支持图片预览）

```vue
<div class="rich-text-content size-large rich-text-clickable" 
     @click="handleImageClick"
     v-html="content">
</div>
```

### 紧凑型布局（用于表格、列表）

```vue
<div class="rich-text-content rich-text-compact size-small" 
     v-html="item.content">
</div>
```

## 最佳实践

### 1. 选择合适的尺寸

| 场景 | 推荐尺寸 | 说明 |
|------|----------|------|
| 题目内容 | `size-large` 或 `size-xlarge` | 需要较大图片展示 |
| 答案选项 | `size-medium` | 中等尺寸，适合选项展示 |
| 表格内容 | `size-small` + `rich-text-compact` | 紧凑显示，节省空间 |
| 列表项 | `size-small` | 小尺寸，适合列表 |

### 2. 图片预览功能

需要图片预览功能时，添加 `rich-text-clickable` 类：

```vue
<div class="rich-text-content rich-text-clickable" 
     @click="handleContentClick"
     v-html="content">
</div>
```

### 3. 自定义样式

如需自定义特定样式，在组件内覆盖：

```css
.my-custom-content :deep(img) {
  border: 2px solid #409eff;
  box-shadow: 0 2px 12px rgba(64, 158, 255, 0.3);
}
```
