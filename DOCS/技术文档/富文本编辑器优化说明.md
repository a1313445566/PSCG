# 富文本编辑器优化说明

## 优化内容

### 1. 扩展工具栏功能

#### 完整版工具栏（`toolbar-mode="full"`）
适用于题目内容编辑，包含以下工具：

**基础格式**
- 加粗、斜体、下划线、删除线
- 上标、下标（数学题必备）

**段落格式**
- 标题（H1-H6）
- 字体大小（小、正常、大、超大）
- 字体选择
- 文字颜色、背景色

**列表和缩进**
- 有序列表
- 无序列表
- 增加缩进、减少缩进

**对齐方式**
- 左对齐、居中、右对齐、两端对齐

**高级功能**
- 引用块
- 代码块
- 插入链接
- 插入图片
- 插入视频
- 清除格式

#### 基础版工具栏（`toolbar-mode="basic"`）
适用于答案选项编辑，精简但实用：
- 基础格式：加粗、斜体、下划线
- 上下标（数学答案）
- 颜色设置
- 列表
- 插入链接、图片
- 清除格式

#### 精简版工具栏（`toolbar-mode="minimal"`）
适用于简单输入：
- 基础格式：加粗、斜体、下划线
- 颜色设置
- 清除格式

### 2. 工具栏中文提示

所有工具栏按钮和下拉选择器都添加了中文提示：
- 鼠标悬停显示中文说明
- 提升用户体验
- 无需记忆图标含义

### 3. 样式优化

**工具栏样式**
- 优化按钮悬停效果
- 高亮显示当前格式
- 改进下拉选择器样式

**编辑器样式**
- 增加最小高度（150px）
- 优化行高（1.6）
- 焦点状态高亮

**响应式优化**
- 工具栏自动换行
- 适配不同屏幕尺寸

## 使用方法

### 题目内容编辑（完整版）

```vue
<QuillEditor
  v-model="form.content"
  toolbar-mode="full"
  :options="{
    placeholder: '输入题目内容'
  }"
/>
```

### 答案选项编辑（基础版）

```vue
<QuillEditor
  v-model="form.option"
  toolbar-mode="basic"
  :options="{
    placeholder: '输入答案内容'
  }"
/>
```

### 简单输入（精简版）

```vue
<QuillEditor
  v-model="form.note"
  toolbar-mode="minimal"
  :options="{
    placeholder: '输入备注'
  }"
/>
```

## 兼容性

- ✅ 完全向后兼容
- ✅ 保留原有的 `options` prop 自定义能力
- ✅ 默认使用完整版工具栏

## 注意事项

1. **图片上传**：保留原有的图片上传功能（粘贴、拖拽、工具栏按钮）
2. **自定义配置**：如果同时使用 `toolbarMode` 和 `options.modules.toolbar`，后者会覆盖前者
3. **性能优化**：工具栏配置预定义，避免重复创建

## 快捷键

Quill 编辑器支持以下快捷键：
- **Ctrl+B** - 加粗
- **Ctrl+I** - 斜体
- **Ctrl+U** - 下划线
- **Ctrl+Z** - 撤销
- **Ctrl+Y** - 重做
- **Ctrl+Shift+S** - 删除线
- **Ctrl+K** - 插入链接
