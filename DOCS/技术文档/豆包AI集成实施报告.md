## 🔧 技术规范遵循

### ✅ 数据库规范
- 使用 MySQL 语法（`AUTO_INCREMENT`, `VARCHAR`, `INT`）
- 使用 `db.get/query/run` 方法
- 参数化查询防止 SQL 注入

### ✅ API 调用规范
- 使用 `api` 封装，禁止原生 fetch
- 自动重试、防抖、超时处理
- 统一错误处理

### ✅ 消息提示规范
- 使用 `message` 封装，禁止 Element Plus
- 统一的消息提示样式

### ✅ HOOK 使用规范
- 正确使用 `useLoading`
- 组件卸载时清理资源

### ✅ 图标导入规范
- 所有图标从 `@element-plus/icons-vue` 导入

## 🚀 使用指南

### 管理员配置步骤

1. **登录后台管理**
   ```
   访问: http://your-server:3001
   ```

2. **进入基础设置**
   ```
   侧边栏 → 基础设置 → AI 智能分析配置
   ```

3. **配置豆包 AI**
   - 输入 API Key（从豆包控制台获取）
   - 输入 API URL（默认已填）
   - 输入模型 Endpoint ID
   - 启用 AI 功能
   - 启用缓存（推荐）

4. **测试连接**
   ```
   点击"测试连接"按钮验证配置
   ```

5. **保存配置**
   ```
   点击"保存配置"完成设置
   ```

### 教师使用步骤

1. **进入数据分析页面**
   ```
   侧边栏 → 数据分析
   ```

2. **使用 AI 分析**
   - 在 AI 智能分析面板输入问题
   - 或点击示例问题快速填充
   - 点击"开始分析"

3. **查看结果**
   - 查看分析报告
   - 复制结果
   - 导出报告
