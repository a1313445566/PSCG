# 项目介绍

## 项目概述

PSCG (Primary School Comprehensive Quiz) 是一个面向小学教育的智能题库系统，采用前后端分离架构，为学生提供答题练习、错题管理、学习分析等功能，为教师提供题目管理、数据统计、AI辅助等管理功能。

## 核心功能模块

### 1. 用户系统
- 学生登录：使用学号、年级、班级进行身份验证
- 管理员登录：独立的管理员认证体系
- JWT Token 认证机制

### 2. 学科系统
- 支持多学科（数学、语文、英语等）
- 每个学科包含多个子分类
- 学科和子分类可配置排序

### 3. 题目系统
- 支持单选题和多选题
- 题目包含内容、选项、答案、难度等属性
- 支持图片和音频附件
- 批量导入功能

### 4. 答题系统
- 随机抽题机制
- 选项乱序功能
- 答题计时
- 错题自动收集
- 错题巩固模式

### 5. 统计系统
- 个人学习数据统计
- 答题正确率分析
- 学习进度跟踪
- 排行榜功能

### 6. AI 助手系统
- 集成 AI 聊天功能
- 支持多模型配置
- Token 使用统计
- 智能学习分析

## 技术架构

### 前端架构
- **框架**: Vue 3 + Composition API
- **UI 库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router (Hash 模式)
- **构建工具**: Vite
- **可视化**: VisActor VChart

### 后端架构
- **运行时**: Node.js
- **框架**: Express.js
- **数据库**: MySQL
- **认证**: JWT
- **文件上传**: Multer
- **缓存**: 内存缓存

### 数据库设计
- **关系型数据库**: MySQL 8.0+
- **主要表**: 
  - 用户相关：users, grades, classes
  - 学科相关：subjects, subcategories
  - 题目相关：questions, question_attempts
  - 答题相关：quiz_sessions, quiz_attempts, answer_records
  - 错题相关：error_collection
  - AI 相关：admin_credentials, ai_models, chat_sessions, chat_messages, chat_summaries, token_usage, cache_hits
  - 文件管理：file_hashes

## 项目特点

1. **前后端分离**: 清晰的职责划分，便于开发和维护
2. **组件化开发**: 前端组件化，提高代码复用性
3. **响应式设计**: 适配不同屏幕尺寸
4. **数据缓存**: 后端缓存常用数据，提高性能
5. **AI 集成**: 支持 AI 聊天和智能分析
6. **错题巩固**: 自动收集错题，支持针对性复习

## 适用场景

- 小学课后练习
- 课堂测验
- 错题复习
- 学习进度跟踪
- 教学数据分析
