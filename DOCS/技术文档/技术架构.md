#### 核心特性

1. **变量系统**
   - 颜色变量：`$primary-color: #ff6b6b`
   - 间距变量：`$spacing-md: 16px`
   - 断点变量：`$breakpoint-md: 768px`

2. **混合宏系统**
   - 响应式：`@include breakpoint('md')`
   - 布局：`@include flex-center`、`@include flex-between`
   - 卡片：`@include card-style`

3. **Element Plus 主题**
   - 自动应用项目主色 `#ff6b6b`
   - 统一视觉风格

4. **迁移状态**
   - ✅ SCSS 基础设施已创建
   - ⚠️ 待接入项目（main.scss 未引入 main.js）
   - 📝 详见《SCSS重构需求方案.md》

## 数据库架构

### 核心表结构

#### users - 用户表
```sql
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  student_id VARCHAR(20) NOT NULL COMMENT '学号',
  name VARCHAR(50) NOT NULL COMMENT '姓名',
  grade VARCHAR(20) NOT NULL COMMENT '年级',
  class VARCHAR(20) NOT NULL COMMENT '班级',
  points INT DEFAULT 0 COMMENT '积分',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  UNIQUE KEY unique_student (student_id, grade, class)
);
```

#### subjects - 学科表
```sql
CREATE TABLE subjects (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(50) NOT NULL COMMENT '学科名称',
  icon_index INT DEFAULT 0 COMMENT '图标索引',
  sort_order INT DEFAULT 0 COMMENT '排序',
  show_in_history_quiz TINYINT(1) DEFAULT 0 COMMENT '是否在历史答题中显示',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

#### subcategories - 子分类表
```sql
CREATE TABLE subcategories (
  id INT PRIMARY KEY AUTO_INCREMENT,
  subject_id INT NOT NULL COMMENT '学科 ID',
  name VARCHAR(100) NOT NULL COMMENT '分类名称',
  icon_index INT DEFAULT 0 COMMENT '图标索引',
  difficulty INT DEFAULT 1 COMMENT '难度',
  sort_order INT DEFAULT 0 COMMENT '排序',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_subject_id (subject_id),
  FOREIGN KEY (subject_id) REFERENCES subjects(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

#### questions - 题目表
```sql
CREATE TABLE questions (
  id INT PRIMARY KEY AUTO_INCREMENT,
  subject_id INT NOT NULL COMMENT '学科 ID',
  subcategory_id INT NOT NULL COMMENT '子分类 ID',
  content TEXT NOT NULL COMMENT '题目内容',
  type ENUM('single', 'multiple') DEFAULT 'single' COMMENT '题型',
  options TEXT NOT NULL COMMENT '选项 (JSON 格式)',
  correct_answer TEXT NOT NULL COMMENT '正确答案',
  explanation TEXT COMMENT '答案解析',
  difficulty INT DEFAULT 1 COMMENT '难度',
  audio_url VARCHAR(255) COMMENT '音频 URL',
  image_url VARCHAR(255) COMMENT '图片 URL',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_subject_id (subject_id),
  INDEX idx_subcategory_id (subcategory_id),
  INDEX idx_difficulty (difficulty),
  FOREIGN KEY (subject_id) REFERENCES subjects(id) ON DELETE CASCADE,
  FOREIGN KEY (subcategory_id) REFERENCES subcategories(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

#### quiz_sessions - 答题会话表

```sql
CREATE TABLE quiz_sessions (
```

```sql
  message_end_id INT NOT NULL COMMENT '结束消息 ID',
  summary TEXT NOT NULL COMMENT '摘要内容',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  INDEX idx_session_id (session_id),
  FOREIGN KEY (session_id) REFERENCES chat_sessions(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```
