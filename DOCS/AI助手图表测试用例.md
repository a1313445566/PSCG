# AI助手图表功能测试用例

## 测试前准备

1. 确保后端服务已启动（端口 3001）
2. 确保前端服务已启动（端口 5173）
3. 确保数据库中有学生数据（当前主要在4年级）

## 测试用例

### 用例1：班级正确率对比（柱状图）

**测试问题**：
```
查看4年级9班学生的正确率排名，用柱状图展示
```

**预期结果**：
- ✅ 显示学生排名表格
- ✅ 生成柱状图
- ✅ 图表标题为"学生正确率对比"
- ✅ X轴为学生姓名，Y轴为正确率

**验证点**：
- MessageList控制台无错误
- 图表正常渲染
- 数据与表格一致

## 已知限制

1. **数据量限制**：建议单图表不超过100个数据点
2. **图表类型**：目前支持柱状图、折线图、饼图
3. **响应式**：最小宽度400px
4. **导出**：暂不支持导出功能
