## 1. 需求概述

### 1.1 背景

当前后台管理系统使用顶部标签页（`el-tabs`）布局，随着功能模块增多，导航标签数量增加，导致：
- 顶部空间拥挤，标签显示不完整
- 功能模块缺乏分组，用户难以快速定位
- 用户体验不佳，操作效率较低

### 1.2 目标

将顶部标签页布局改造为**左右分栏布局**：
- **左侧**: 可折叠的侧边导航栏，支持分组和图标
- **右侧**: 主内容区域，自适应宽度
- **顶部**: 简化的标题栏，显示当前位置和快捷操作

### 1.3 预期效果

```
┌──────────────────────────────────────────────────────────────┐
│  题库管理系统    首页 / 题目管理     [刷新] [返回] [退出]      │
├────────────┬─────────────────────────────────────────────────┤
│            │                                                 │

│  ├ 📚 题目  │    │         题目管理内容区域             │     │
```

- 移除管理员名称显示（移到个人信息弹窗或设置中）

- 自适应宽度，根据侧边栏宽度动态调整
- 支持内容区域全屏模式
- 保持现有功能组件不变，仅调整布局容器

| 趋势图表 | 近7天/30天答题趋势 |
| 热门学科 | 各学科答题分布 |
| 最近活动 | 最近10条答题记录 |

## 5. 增量开发策略

| 原则 | 说明 |
|------|------|
| **增量迭代** | 按优先级分阶段开发，每阶段完成后验证再进入下一阶段 |
| **最小影响** | 每次改动尽量小，确保现有功能不受影响 |
| **可回滚** | 每个增量完成后可独立部署，出问题可快速回滚 |
| **渐进增强** | 先实现核心功能，再添加高级特性 |

## 6. 实现步骤

### 步骤 1: 创建布局状态 Hook

创建 `src/composables/useAdminLayout.js`：

```javascript
import { ref, computed } from 'vue'

const isCollapse = ref(false)
const sidebarWidth = ref(220)
const activeMenu = ref('dashboard')

export function useAdminLayout() {
  const toggleCollapse = () => {
    isCollapse.value = !isCollapse.value
  }
  
  const setActiveMenu = (key) => {
    activeMenu.value = key
  }
  
  const expandedWidth = computed(() => sidebarWidth.value)
  const collapsedWidth = 64
  
  return {
    isCollapse,
    sidebarWidth,
    activeMenu,
    toggleCollapse,
    setActiveMenu,
    expandedWidth,
    collapsedWidth
  }
}
```

### 6.2 后端实现

在 `routes/dashboard.js` 中实现：

```javascript
const express = require('express');
const router = express.Router();
const adminAuth = require('../middleware/adminAuth');
const db = require('../services/database');

router.get('/stats', adminAuth, async (req, res) => {
  try {
    // 统计数据查询
    const totalQuestions = await db.get('SELECT COUNT(*) as count FROM questions');
    const totalUsers = await db.get('SELECT COUNT(*) as count FROM users');
    // ... 更多统计
    
    res.json({
      totalQuestions: totalQuestions.count,
      totalUsers: totalUsers.count,
      // ...
    });
  } catch (error) {
    console.error('[Dashboard] 获取统计数据失败:', error);
    res.status(500).json({ error: '获取统计数据失败' });
  }
});

module.exports = router;
```
