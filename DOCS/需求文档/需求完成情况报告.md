## 🐛 问题修复记录 (2026-03-28)

### 1. 错题练习 404 问题
- **问题**: 点击错题练习报 404 错误
- **原因**: 前端路径 `error-practice` 与后端路由 `error-collection` 不匹配
- **修复**: 统一使用 `error-collection` 路径
- **文件**: `src/views/ErrorBookView.vue`

### 2. 学习进度 500 错误
- **问题**: API `/api/learning-progress/user/:id` 返回 500
- **原因**: 
  - 数据库表缺少 `total_attempts`、`correct_attempts` 字段
  - 字段名不一致 `last_practiced_at` vs `last_practiced`
- **修复**: 
  - 添加缺失字段
  - 统一字段名为 `last_practiced`
- **文件**: `routes/learning-progress.js`

### 3. VChart 图表渲染错误
- **问题**: `TypeError: i.replace is not a function`
- **原因**: 图表数据字段类型不是字符串
- **修复**: 所有图表数据字段强制类型转换
  - 字符串: `String(value || '')`
  - 数值: `Number(value || 0)`
- **文件**: 
  - `LearningReportView.vue`
  - `AnswerHistoryView.vue`
  - `DashboardView.vue`
  - `AnalysisCharts.vue`

### 4. withLoading 使用错误
- **问题**: `TypeError: r is not a function`
- **原因**: `withLoading(fetchReport())` 传递了 Promise 而非函数
- **修复**: `withLoading(fetchReport)` 传递函数引用
- **文件**: `src/views/LearningReportView.vue`

### 5. API 路径错误
- **问题**: `/api/users/225/stats` 返回 404
- **原因**: 路由顺序导致 `/users/:id` 先匹配
- **修复**: 前端改为 `/api/users/stats/:userId`
- **文件**: `src/views/LearningReportView.vue`

### 6. VChart 循环依赖警告
- **问题**: 构建时报循环依赖警告
- **修复**: Vite 配置 `manualChunks` 将 VChart 打包到独立 chunk
- **文件**: `vite.config.js`

### 7. 学习进度无数据
- **问题**: 学习进度页面无数据
- **原因**: 表新建，缺少历史数据同步逻辑
- **修复**: 
  - 添加答题提交时自动更新学习进度
  - 执行历史数据同步（354 条记录）
- **文件**: `routes/quiz.js`
