```javascript
const hasAnsweredAll = computed(() => {
  return currentQuestions.value.every(question => {
    const answer = userAnswers.value[question.id]

    // 阅读理解题：检查所有小题是否都已作答
    if (question.type === 'reading') {
      if (!answer || typeof answer !== 'object') return false
      const options = typeof question.options === 'string'
        ? JSON.parse(question.options)
        : question.options
      return options.every(sq =>
        answer[String(sq.order || options.indexOf(sq) + 1)] !== undefined
      )
    }
    // ...
  })
})
```

**优点**：
- 正确识别阅读理解题的答案格式（对象）
- 遍历所有小题检查是否已作答
- 兼容 `sq.order` 不存在的情况（使用 `indexOf + 1` 作为备用）

### 3.3 XSS 递归过滤验证（#11）

**需求文档位置**：第 620-653 行

**评估**：✅ 完整覆盖

**修复方案质量**：⭐⭐⭐⭐⭐

```javascript
const subQuestions = computed(() => {
  let options = props.question.options
  if (typeof options === 'string') {
    try { options = JSON.parse(options) } catch (e) { return [] }
  }
  if (!Array.isArray(options)) return []

  // 递归过滤所有富文本字符串
  return options.map(sq => ({
    ...sq,
    content: xssFilter.deepSanitize(sq.content),
    options: (sq.options || []).map(opt => xssFilter.deepSanitize(opt)),
    explanation: sq.explanation ? xssFilter.deepSanitize(sq.explanation) : ''
  }))
})
```

**优点**：
- 对每个富文本字段单独调用 `deepSanitize`
- 覆盖了 `content`、`options`、`explanation` 三个富文本字段
- 处理了 `explanation` 可能为空的情况

**补充建议**：
- 建议在后端 `questions.js` 也增加相同的递归过滤逻辑，作为双重保障

## 五、仍需注意的问题

### 5.1 后端 XSS 过滤递归深度

**位置**：`routes/questions.js` 第 320-335 行

**现状**：需求文档第 50 行提到"现有 `routes/questions.js` 第320-335行已对 options 做递归 XSS 过滤"，但实际代码只处理了一层对象嵌套：

```javascript
if (typeof opt === 'object') {
  return Object.keys(opt).reduce((acc, key) => {
    acc[key] = typeof opt[key] === 'string' ? xssFilter.deepSanitize(opt[key]) : opt[key]
    return acc
  }, {})
}
```

**问题**：阅读理解题的 `options` 结构为 `[{order, content, options, explanation}]`，其中 `options` 是字符串数组，当前代码不会递归处理数组内的字符串。

**建议**：在后端也实现递归过滤函数：

```javascript
const sanitizeDeep = (obj) => {
  if (typeof obj === 'string') {
    return xssFilter.deepSanitize(obj)
  }
  if (Array.isArray(obj)) {
    return obj.map(item => sanitizeDeep(item))
  }
  if (typeof obj === 'object' && obj !== null) {
    return Object.keys(obj).reduce((acc, key) => {
      acc[key] = sanitizeDeep(obj[key])
      return acc
    }, {})
  }
  return obj
}
```
