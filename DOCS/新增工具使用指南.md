## 📋 新增工具清单

### 1. 学生预警工具 (`query_student_alerts`)
**小学教育最刚需工具！**

**功能**：
- ✅ 识别连续错误超过 3 次的学生
- ✅ 识别正确率突然下降超过 20% 的学生
- ✅ 识别连续 3 天未登录的学生
- ✅ 识别答题量明显少于平均的学生

**使用示例**：
```
"最近一周有哪些学生需要重点关注？"
"找出连续错误超过3次的学生"
"查找正确率下降超过20%的学生"
```

**参数**：
- `grade`: 年级（可选）
- `className`: 班级（可选）
- `alertType`: 预警类型（`all`, `consecutive_errors`, `accuracy_drop`, `inactive`, `low_activity`）
- `days`: 统计天数（默认7天）
- `threshold`: 阈值

---

### 2. 知识点掌握度工具 (`query_knowledge_point_mastery`)
**精准教学核心工具！**

**功能**：
- ✅ 查询学生在具体知识点（子分类）上的掌握度
- ✅ 列出全班掌握度低于 70% 的知识点
- ✅ 找出每个学生的"最薄弱知识点 Top 3"

**使用示例**：
```
"张三在'字形辨析'这个知识点掌握得怎么样？"
"班上哪些学生还没掌握'进位加法'？"
"查看全班在各个知识点的掌握情况"
```

**参数**：
- `studentId`: 学生ID（查询单个学生）
- `grade`: 年级（查询全班）
- `className`: 班级（查询全班）
- `subjectId`: 学科ID（可选）
- `subcategoryId`: 知识点ID（可选）
- `masteryThreshold`: 掌握度阈值（默认70）

---

### 3. 个性化推荐工具 (`query_personalized_recommendations`)
**分层教学必备工具！**

**功能**：
- ✅ 根据学生薄弱点推荐题目
- ✅ 自适应难度（根据正确率调整）
- ✅ 为学优生推荐挑战题，为学困生推荐基础题

**使用示例**：
```
"给张三推荐10道适合他水平的题目"
"推荐5道适合小明的'进位加法'练习题"
"根据薄弱点推荐题目"
```

**参数**：
- `studentId`: 学生ID（必填）
- `subjectId`: 学科ID（可选）
- `count`: 推荐数量（默认10）
- `difficultyRange`: 难度策略（`adaptive`, `easy`, `medium`, `hard`）
- `focusWeak`: 是否聚焦薄弱知识点（默认true）

---

### 4. 班级对比工具 (`query_class_comparison`)
**教育管理工具！**

**功能**：
- ✅ 对比多个班级的学习表现
- ✅ 按正确率、进步幅度、活跃度排名
- ✅ 识别优秀班级的教学方法

**使用示例**：
```
"对比一班和二班的数学成绩"
"查看三年级各班的正确率排名"
"对比各班的进步幅度"
```

**参数**：
- `grade`: 年级（必填）
- `classes`: 班级列表（如 `["一班", "二班"]`）
- `subjectId`: 学科ID（可选）
- `compareBy`: 对比维度（`accuracy`, `progress`, `activity`, `all`）

---

### 5. 学习轨迹追踪工具 (`query_learning_path`)
**家校沟通工具！**

**功能**：
- ✅ 查看学生最近 N 天的答题趋势
- ✅ 对比学生与班级平均水平的差距
- ✅ 标记学生的"高光时刻"（连续答对等）
- ✅ 生成学习评语（AI 生成）

**使用示例**：
```
"张三最近一周学得怎么样？"
"查看小明本月的学习轨迹"
"生成张三的学习报告"
```

**参数**：
- `studentId`: 学生ID（必填）
- `days`: 追踪天数（默认30天）
- `detail`: 详情级别（`summary`, `timeline`, `trend`, `all`）
