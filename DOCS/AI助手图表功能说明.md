# AI助手图表功能说明

## 功能概述

AI助手现在支持自动生成可视化图表，让数据分析更直观。

## 支持的图表类型

### 1. 柱状图（Bar Chart）
**适用场景**：对比数据、排名展示、班级对比

**示例配置**：
```json
{
  "type": "bar",
  "data": [{
    "id": "correctRate",
    "values": [
      { "x": "小明", "y": 85 },
      { "x": "小红", "y": 92 },
      { "x": "小刚", "y": 78 }
    ]
  }],
  "title": { "text": "学生正确率对比" },
  "axes": [
    { "orient": "bottom", "title": "学生姓名" },
    { "orient": "left", "title": "正确率(%)" }
  ]
}
```

### 2. 折线图（Line Chart）
**适用场景**：趋势分析、学习曲线、时间序列

**示例配置**：
```json
{
  "type": "line",
  "data": [{
    "id": "trend",
    "values": [
      { "x": "第1周", "y": 70 },
      { "x": "第2周", "y": 75 },
      { "x": "第3周", "y": 82 },
      { "x": "第4周", "y": 88 }
    ]
  }],
  "title": { "text": "学习趋势" },
  "axes": [
    { "orient": "bottom", "title": "时间" },
    { "orient": "left", "title": "正确率(%)" }
  ]
}
```

### 3. 饼图（Pie Chart）
**适用场景**：占比分析、错题类型分布、知识点掌握情况

**示例配置**：
```json
{
  "type": "pie",
  "data": [{
    "id": "errorTypes",
    "values": [
      { "x": "计算错误", "y": 30 },
      { "x": "概念理解", "y": 25 },
      { "x": "审题失误", "y": 20 },
      { "x": "其他", "y": 25 }
    ]
  }],
  "title": { "text": "错题类型分布" }
}
```

## 使用方法

**示例问题**：

- "分析小明的学习趋势"
- "统计错题类型分布"

AI会自动：

在问题中明确要求图表：

**示例问题**：

- "画一个折线图显示小明的学习趋势"
- "用饼图分析错题类型"

## 图表渲染流程

1. **AI分析数据** → 识别适合的图表类型
2. **生成配置** → 创建符合VChart规范的JSON
3. **前端渲染** → MessageList组件自动识别并渲染

## 图表配置规范

### 基本结构
```json
{
  "type": "图表类型",
  "data": [{ "id": "系列名", "values": [...] }],
  "title": { "text": "图表标题" },
  "axes": [{ "orient": "轴方向", "title": "轴标题" }]
}
```

### 数据格式
```json
{
  "values": [
    { "x": "分类1", "y": 数值1 },
    { "x": "分类2", "y": 数值2 }
  ]
}
```

### 注意事项
- ✅ 必须使用双引号（JSON标准）
- ✅ type字段必须存在
- ✅ data数组必须包含values
- ❌ 不要使用单引号
- ❌ 不要有多余的逗号

## 常见问题

### Q1: 图表没有显示？
**原因**：AI可能没有生成图表配置
**解决**：明确要求"用图表展示"

### Q2: 图表显示错误？
**原因**：JSON格式不正确
**解决**：MessageList会自动尝试修复常见错误

### Q3: 如何让图表更美观？
**建议**：
- 数据点不超过10个（柱状图）
- 类别不超过6个（饼图）
- 时间跨度合理（折线图）

## 技术细节

### 前端实现
- **组件**：`MessageList.vue`
- **库**：VChart (@visactor/vchart)
- **识别方式**：`<code class="language-json-chart">`
- **渲染时机**：Markdown解析后延迟50ms

### 后端实现
- **文件**：`services/chat/agent.js`
- **提示词**：`generateSystemPrompt()`
- **模型要求**：支持Function Calling

### 图表配置验证
MessageList会自动修复常见JSON错误：
- 移除末尾多余逗号
- 单引号转双引号
- 验证必需字段
