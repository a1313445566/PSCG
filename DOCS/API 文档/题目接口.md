# 题目管理 API 文档

## 基础信息

- **基础路径**: `/api/questions`
- **认证方式**: 管理员需要 JWT Token（通过 Authorization 头）
- **响应格式**: JSON

## API 接口列表

### 1. 获取子分类列表

**请求**
```
GET /api/questions/subcategories
```

**查询参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| subjectId | number | 否 | 学科 ID，用于筛选 |

**响应示例**
```json
[
  {
    "id": 1,
    "name": "选择题",
    "subjectId": 1,
    "subjectName": "数学",
    "questionCount": 100
  }
]
```

### 2. 获取子分类统计数据

**请求**
```
GET /api/questions/subcategories/stats
```

**查询参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| subjectId | number | 否 | 学科 ID |

**响应示例**
```json
{
  "1": {
    "questionCount": 100,
    "avgDifficulty": 2.5
  },
  "2": {
    "questionCount": 50,
    "avgDifficulty": 3.0
  }
}
```

### 3. 获取题目列表

**请求**
```
GET /api/questions
```

**查询参数**
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| subjectId | number | 否 | - | 学科 ID |
| subcategoryId | number | 否 | - | 子分类 ID |
| type | string | 否 | - | 题型 (single/multiple) |
| difficulty | number | 否 | - | 难度 (1-5) |
| keyword | string | 否 | - | 关键词搜索 |
| page | number | 否 | 1 | 页码 |
| limit | number | 否 | 20 | 每页数量 |
| excludeContent | string | 否 | 'false' | 是否排除题目内容 |

**响应示例**
```json
{
  "data": [
    {
      "id": 1,
      "subjectId": 1,
      "subjectName": "数学",
      "subcategoryId": 1,
      "subcategoryName": "选择题",
      "type": "single",
      "difficulty": 2,
      "options": "{\"A\":\"选项 1\",\"B\":\"选项 2\"}",
      "correctAnswer": "A",
      "imageUrl": "/images/image-123.jpg",
      "audioUrl": "/audio/audio-456.mp3",
      "createdAt": "2024-01-01T00:00:00.000Z"
    }
  ],
  "total": 100,
  "page": 1,
  "limit": 20,
  "totalPages": 5
}
```

### 4. 获取单个题目详情

**请求**
```
GET /api/questions/:id
```

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 题目 ID |

**响应示例**
```json
{
  "id": 1,
  "subjectId": 1,
  "subjectName": "数学",
  "subcategoryId": 1,
  "subcategoryName": "选择题",
  "content": "题目内容",
  "type": "single",
  "options": {
    "A": "选项 1",
    "B": "选项 2",
    "C": "选项 3",
    "D": "选项 4"
  },
  "correctAnswer": "A",
  "explanation": "答案解析",
  "difficulty": 2,
  "imageUrl": "/images/image-123.jpg",
  "audioUrl": "/audio/audio-456.mp3",
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

### 5. 创建题目

**请求**
```
POST /api/questions
```

**认证**: 需要管理员权限

**请求体**
```json
{
  "subjectId": 1,
  "subcategoryId": 1,
  "content": "题目内容",
  "type": "single",
  "options": {
    "A": "选项 1",
    "B": "选项 2",
    "C": "选项 3",
    "D": "选项 4"
  },
  "correctAnswer": "A",
  "explanation": "答案解析",
  "difficulty": 2,
  "imageUrl": "/images/image-123.jpg",
  "audioUrl": "/audio/audio-456.mp3"
}
```

**响应示例**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "subjectId": 1,
    "subcategoryId": 1,
    "content": "题目内容",
    "type": "single",
    "correctAnswer": "A"
  }
}
```

### 6. 更新题目

**请求**
```
PUT /api/questions/:id
```

**认证**: 需要管理员权限

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 题目 ID |

**请求体**
```json
{
  "subjectId": 1,
  "subcategoryId": 1,
  "content": "更新后的题目内容",
  "type": "single",
  "options": {
    "A": "选项 1",
    "B": "选项 2",
    "C": "选项 3",
    "D": "选项 4"
  },
  "correctAnswer": "B",
  "explanation": "更新后的解析",
  "difficulty": 3
}
```

**响应示例**
```json
{
  "success": true,
  "message": "题目更新成功",
  "data": {
    "id": 1,
    "subjectId": 1,
    "subcategoryId": 1,
    "content": "更新后的题目内容",
    "type": "single",
    "correctAnswer": "B"
  }
}
```

### 7. 删除题目

**请求**
```
DELETE /api/questions/:id
```

**认证**: 需要管理员权限

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number | 是 | 题目 ID |

**响应示例**
```json
{
  "success": true,
  "message": "题目删除成功"
}
```

### 8. 批量导入题目

**请求**
```
POST /api/questions/batch-import
```

**认证**: 需要管理员权限

**请求体**
```json
{
  "subjectId": 1,
  "subcategoryId": 1,
  "questions": "1. 题目内容 1(A)\nA. 选项 1\nB. 选项 2\n\n2. 题目内容 2(B)\nA. 选项 1\nB. 选项 2"
}
```

**响应示例**
```json
{
  "success": true,
  "message": "批量导入成功",
  "data": {
    "total": 2,
    "success": 2,
    "failed": 0,
    "errors": []
  }
}
```

## 错误响应

### 400 Bad Request
```json
{
  "error": "请求参数错误"
}
```

### 401 Unauthorized
```json
{
  "error": "未授权访问"
}
```

### 404 Not Found
```json
{
  "error": "题目不存在"
}
```

### 500 Internal Server Error
```json
{
  "error": "服务器内部错误"
}
```

## 注意事项

1. **题目格式**: options 字段支持 JSON 字符串或对象格式
2. **文件上传**: 图片和音频通过 `/api/upload` 接口上传，返回 URL 后保存到题目
3. **批量导入**: 题目文本格式必须规范，否则解析会失败
4. **删除约束**: 删除题目会级联删除相关的答题记录和错题记录
