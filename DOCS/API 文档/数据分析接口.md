# 数据分析 API 文档

## 基础信息

- **基础路径**: `/api/analysis`
- **认证方式**: 学生登录 / 管理员 JWT
- **响应格式**: JSON

---

## API 接口列表

### 1. 获取学习统计

**请求**
```
GET /api/analysis/stats
```

**认证**: 学生登录

**响应示例**
```json
{
  "success": true,
  "data": {
    "totalSessions": 50,
    "totalQuestions": 500,
    "totalCorrect": 425,

    "accuracy": 85.0,

    "totalPoints": 380,
    "errorBookCount": 25,
    "masteredCount": 15
```

```json
}
```

### 3. 获取答题趋势

**请求**
```
GET /api/analysis/trend
```

**认证**: 学生登录

**查询参数**
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| days | number | 否 | 30 | 天数 |
| subjectId | number | 否 | - | 学科 ID |

**响应示例**
```json
{
  "success": true,
  "data": {

        "date": "2024-01-01",
```

## 使用示例

### 前端调用示例
```javascript
// 获取学习统计
const getStats = async () => {
  const response = await fetch('/api/analysis/stats')
  const data = await response.json()
  return data.data
}

// 获取学科分析
const getSubjectAnalysis = async () => {
  const response = await fetch('/api/analysis/subjects')
  const data = await response.json()
  return data.data
}

// 获取答题趋势
const getTrend = async (days = 30) => {
  const response = await fetch(`/api/analysis/trend?days=${days}`)
  const data = await response.json()
  return data.data
}
```

## 注意事项

1. **数据更新**: 统计数据准实时更新，可能存在短暂延迟
2. **时间范围**: 趋势分析支持自定义时间范围
3. **数据隐私**: 学生只能查看自己的数据
4. **管理员权限**: 管理员可以查看全校和班级数据
5. **性能优化**: 大数据量时使用缓存和分页
