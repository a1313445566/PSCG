# 系统设置 API 文档

## 基础信息

- **基础路径**: `/api/settings`
- **认证方式**: 管理员 JWT Token
- **响应格式**: JSON

---

## API 接口列表

### 1. 获取系统设置

**请求**
```
GET /api/settings
```

**认证**: 需要管理员 Token

**响应示例**
```json
{
  "success": true,
  "data": {
    "systemName": "PSCG 学习系统",
    "itemsPerPage": 20,
    "quizTimeLimit": 30,
    "enableRandomOptions": true,
    "enableBehaviorTracking": true,
    "allowMultiDeviceLogin": true,
    "tokenExpiresIn": "24h",
    "maintenanceMode": false
  }
}
```

### 2. 更新系统设置

## 使用示例

### 前端调用示例
```javascript
// 获取系统设置
const getSettings = async () => {
  const response = await fetch('/api/settings', {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  })
  const data = await response.json()
  return data.data
}

// 更新系统设置
const updateSettings = async (settings) => {
  const response = await fetch('/api/settings', {
    method: 'PUT',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${token}`
    },
    body: JSON.stringify(settings)
  })
  const data = await response.json()
  return data
}
```

## 注意事项

1. **权限控制**: 只有管理员可以修改系统设置
2. **即时生效**: 设置修改后立即生效
3. **数据验证**: 所有设置值都会进行验证
4. **默认值**: 可以重置为默认值
5. **审计日志**: 设置修改会记录日志
