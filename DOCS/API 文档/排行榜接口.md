# 排行榜 API 文档

## 基础信息

- **基础路径**: `/api/leaderboard`
- **认证方式**: 学生登录
- **响应格式**: JSON

---

### 2. 获取全局排行榜

**请求**
```
GET /api/leaderboard/global
```

**查询参数**
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | number | 否 | 1 | 页码 |
| limit | number | 否 | 50 | 每页数量 |
| grade | string | 否 | - | 年级筛选 |

**响应示例**
```json
{
  "success": true,
  "data": {
    "rankings": [
      {
        "rank": 1,
        "studentId": "01",
        "name": "张三",
        "grade": "六年级",

        "totalQuestions": 300,
        "accuracy": 90.5
      }
    ],
    "total": 500,
    "page": 1,
    "limit": 50,

}
```

### 4. 获取学科排行榜

**请求**
```
GET /api/leaderboard/subject/:subjectId
```

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| subjectId | number | 是 | 学科 ID |

**查询参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| scope | string | 否 | 范围 (class/global) |
| grade | string | 否 | 年级 |
| class | string | 否 | 班级 |

**响应示例**
```json
{
  "success": true,
  "data": {
    "subjectId": 1,
    "subjectName": "数学",

      {
        "rank": 1,

  }

```

## 排名规则

### 主要排序依据
1. **积分** - 从高到低排序
2. **答题总数** - 积分相同时，答题多的靠前
3. **正确率** - 答题总数相同时，正确率高的靠前

### 积分计算
```
普通答题：
  基础分 = 答对题数 - 答错题数
  全对奖励 = 基础分 × 2

错题巩固：
  每道题累计答对 3 次 = +1 分
```

## 使用示例

### 前端调用示例
```javascript
// 获取班级排行榜
const getClassLeaderboard = async (grade, class) => {
  const response = await fetch(
    `/api/leaderboard/class?grade=${encodeURIComponent(grade)}&class=${encodeURIComponent(class)}`
  )
  const data = await response.json()
  return data.data
}

// 获取全局排行榜
const getGlobalLeaderboard = async (page = 1, limit = 50) => {
  const response = await fetch(
    `/api/leaderboard/global?page=${page}&limit=${limit}`
  )
  const data = await response.json()
  return data.data
}

// 获取我的排名
const getMyRank = async () => {
  const response = await fetch('/api/leaderboard/my-rank')
  const data = await response.json()
  return data.data
}
```

## 注意事项

1. **实时更新**: 排行榜数据准实时更新，可能存在短暂延迟
2. **隐私保护**: 只显示基本信息，不显示详细数据
3. **分页查询**: 全局排行榜支持分页，每页默认 50 条
4. **筛选功能**: 支持按年级、学科筛选
5. **并列排名**: 积分相同按规则排序，不占用额外名次
