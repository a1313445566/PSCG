# API 接口概述

## 基础信息

- **基础 URL**: `http://localhost:3001`
- **API 前缀**: `/api`
- **完整示例**: `http://localhost:3001/api/subjects`

## 响应格式

### 学生接口响应格式
大多数学生接口直接返回数据或以下格式：
```json
{
  "success": true,
  "data": {}
}
```

### 管理员接口响应格式
```json
{
  "success": true,
  "message": "操作成功",
  "data": {}
}
```

### 错误响应格式
```json
{
  "error": "错误信息"
}
```

### HTTP 状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 接口分类

### 核心接口
1. **学科接口** - 学科和子分类管理
2. **题目接口** - 题目增删改查
3. **答题接口** - 答题会话和提交
4. **用户接口** - 学生用户管理
5. **排行榜接口** - 成绩排名
6. **数据分析接口** - 学习数据统计

### 高级功能接口
7. **AI 聊天接口** - AI 助手功能
8. **系统设置接口** - 系统配置管理
9. **文件上传接口** - 图片音频上传

### 管理员接口
10. **认证接口** - 管理员认证和用户管理

## 接口列表概览

#### 学科相关
- `GET /api/subjects` - 获取学科列表（含子分类）
- `GET /api/subjects/stats` - 获取学科统计
- `GET /api/subjects/:id/subcategories` - 获取子分类

#### 题目相关
- `GET /api/questions` - 获取题目列表
- `GET /api/questions/subcategories` - 获取子分类列表
- `GET /api/questions/subcategories/stats` - 获取子分类统计
- `POST /api/questions` - 创建题目
- `PUT /api/questions/:id` - 更新题目
- `DELETE /api/questions/:id` - 删除题目

#### 答题相关
- `POST /api/quiz/start` - 开始答题
- `POST /api/quiz/submit` - 提交答案
- `GET /api/quiz/records` - 获取答题记录

#### 排行榜相关
- `GET /api/leaderboard/class` - 班级排行榜
- `GET /api/leaderboard/global` - 全局排行榜
- `GET /api/leaderboard/my-rank` - 我的排名

#### 数据分析相关
- `GET /api/analysis/stats` - 学习统计
- `GET /api/analysis/subjects` - 学科分析
- `GET /api/analysis/trend` - 答题趋势
- `GET /api/analysis/mastery` - 知识点掌握
- `GET /api/analysis/errors` - 错题分析

#### AI 聊天相关
- `GET /api/chat/models` - 获取模型列表
- `POST /api/chat/sessions` - 创建会话
- `POST /api/chat/sessions/:id/messages` - 发送消息
- `GET /api/chat/stats` - Token 使用统计

#### 用户相关
- `GET /api/users` - 获取用户列表
- `GET /api/users/:id` - 获取用户详情
- `POST /api/users` - 创建用户
- `PUT /api/users/:id` - 更新用户
- `DELETE /api/users/:id` - 删除用户

#### 管理员相关
- `GET /api/admin/status` - 检查初始化状态
- `POST /api/admin/init` - 初始化管理员
- `POST /api/admin/login` - 管理员登录
- `PUT /api/admin/password` - 修改密码

#### 系统设置相关
- `GET /api/settings` - 获取系统设置
- `PUT /api/settings` - 更新系统设置

#### 文件上传相关
- `POST /api/upload` - 上传文件

## 认证方式

### 学生认证
- 使用学号 + 姓名 + 年级 + 班级登录
- Token 存储在 localStorage 中
- 有效期 24 小时

### 管理员认证
- 使用用户名 + 密码登录
- 获取 JWT Token
- Token 存储在 sessionStorage 中
- 请求时携带在 Header 中：`Authorization: Bearer {token}`

## 分页参数

列表接口支持分页：

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| page | number | 页码 | 1 |
| limit | number | 每页数量 | 20 |

响应中包含分页信息：
```json
{
  "success": true,
  "data": {
    "items": [],
    "total": 100,
    "page": 1,
    "limit": 20,
    "totalPages": 5
  }
}
```

## 筛选参数

### 题目列表筛选
| 参数 | 类型 | 说明 |
|------|------|------|
| subjectId | number | 学科 ID |
| subcategoryId | number | 子分类 ID |
| type | string | 题型 (single/multiple) |
| difficulty | number | 难度 (1-5) |
| keyword | string | 关键词搜索 |

### 排行榜筛选
| 参数 | 类型 | 说明 |
|------|------|------|
| grade | string | 年级 |
| class | string | 班级 |
| subjectId | number | 学科 ID |

## 使用示例

### 前端调用示例
```javascript
// 获取题目列表
const getQuestions = async (subjectId, page = 1, limit = 20) => {
  const response = await fetch(
    `/api/questions?subjectId=${subjectId}&page=${page}&limit=${limit}`
  )
  const data = await response.json()
  return data.data
}

// 开始答题
const startQuiz = async (subjectId, subcategoryId) => {
  const response = await fetch('/api/quiz/start', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      subjectId,
      subcategoryId,
      questionCount: 10
    })
  })
  const data = await response.json()
  return data.data
}

// 管理员接口调用（需要 Token）
const getStats = async (token) => {
  const response = await fetch('/api/analysis/stats', {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  })
  const data = await response.json()
  return data.data
}
```

## 注意事项

1. **响应格式**: 不同接口可能返回不同格式，请参考具体接口文档
2. **错误处理**: 所有接口都应该处理错误情况
3. **Token 过期**: Token 过期后需要重新登录
4. **速率限制**: AI 接口有速率限制
5. **文件上传**: 支持图片和音频，有大小限制
6. **数据缓存**: 部分接口使用缓存提高性能
