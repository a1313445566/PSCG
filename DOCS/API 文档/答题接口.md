# 答题管理 API 文档

## 基础信息

- **基础路径**: `/api/quiz`
- **认证方式**: 学生登录 (通过 localStorage 中的 studentId)
- **响应格式**: JSON

---

## API 接口列表

### 1. 开始答题会话

**请求**
```
POST /api/quiz/start
```

**请求体**
```json
{
  "subjectId": 1,
  "subcategoryId": 1,
  "questionCount": 10,
  "difficulty": 2
}
```

**响应示例**
```json
{
  "success": true,
  "data": {
    "sessionId": "abc123-def456",
    "questions": [
      {
        "id": 1,
        "content": "题目内容",
        "type": "single",
        "options": {
          "A": "选项 1",
          "B": "选项 2"
        }
      }
    ],
    "expiresAt": "2024-01-01T12:00:00.000Z"
  }
}
```

### 2. 提交答案

**请求**
```
POST /api/quiz/submit
```

**请求体**
```json
{
  "sessionId": "abc123-def456",
  "answers": [
    {
      "questionId": 1,
      "answer": "A",
      "timeSpent": 30
    }
  ]
}
```

**响应示例**
```json
{
  "success": true,
  "data": {
    "results": [
      {
        "questionId": 1,
        "isCorrect": true,
        "correctAnswer": "A",
        "userAnswer": "A"
      }
    ],
    "stats": {
      "totalQuestions": 10,
      "correctCount": 8,
      "accuracy": 80,
      "timeSpent": 300,
      "points": 16
    }
  }
}
```

### 3. 获取答题会话详情

**请求**
```
GET /api/quiz/session/:sessionId
```

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| sessionId | string | 是 | 会话 ID |

**响应示例**
```json
{
  "success": true,
  "data": {
    "sessionId": "abc123-def456",
    "subjectId": 1,
    "subcategoryId": 1,
    "questionCount": 10,
    "submittedCount": 0,
    "createdAt": "2024-01-01T12:00:00.000Z",
    "expiresAt": "2024-01-01T13:00:00.000Z"
  }
}
```

### 4. 取消答题会话

**请求**
```
DELETE /api/quiz/session/:sessionId
```

**路径参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| sessionId | string | 是 | 会话 ID |

**响应示例**
```json
{
  "success": true,
  "message": "答题会话已取消"
}
```

### 5. 获取答题记录

**请求**
```
GET /api/quiz/records
```

**查询参数**
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| subjectId | number | 否 | - | 学科 ID |
| subcategoryId | number | 否 | - | 子分类 ID |
| page | number | 否 | 1 | 页码 |
| limit | number | 否 | 20 | 每页数量 |

**响应示例**
```json
{
  "success": true,
  "data": {
    "records": [
      {
        "id": 1,
        "subjectId": 1,
        "subjectName": "数学",
        "subcategoryId": 1,
        "subcategoryName": "选择题",
        "totalQuestions": 10,
        "correctCount": 8,
        "accuracy": 80,
        "timeSpent": 300,
        "points": 16,
        "createdAt": "2024-01-01T12:00:00.000Z"
      }
    ],
    "total": 50,
    "page": 1,
    "limit": 20,
    "totalPages": 3
```

## 注意事项

1. **会话有效期**: 答题会话默认有效期 1 小时
2. **答案提交**: 必须一次性提交所有答案
3. **防误触**: 提交后无法修改答案
4. **数据保存**: 答案自动保存，但需要手动提交
5. **积分计算**: 根据正确率和难度计算积分

---

## 积分规则

### 普通答题
```
基础分 = 答对题数 - 答错题数
全对奖励 = 基础分 × 2（全部答对时）
```

### 错题巩固
```
每道题累计答对 3 次 = +1 分
再次答错 = 重置计数
```

## 使用示例

### 前端调用示例
```javascript
// 开始答题
const startQuiz = async (subjectId, subcategoryId) => {
  const response = await fetch('/api/quiz/start', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      subjectId,
      subcategoryId,
      questionCount: 10
    })
  })
  
  const data = await response.json()
  return data.data
}

// 提交答案
const submitQuiz = async (sessionId, answers) => {
  const response = await fetch('/api/quiz/submit', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      sessionId,
      answers
    })
  })
  
  const data = await response.json()
  return data.data
}
```
