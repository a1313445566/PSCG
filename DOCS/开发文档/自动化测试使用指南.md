## 一、快速开始

### 1.1 前置条件

1. 开发服务器已启动：`npm run dev`
2. 后端服务器已启动：`npm run server`
3. Playwright 已安装

### 1.2 安装 Playwright 浏览器

```bash
npx playwright install
```

### 1.3 运行所有测试

```bash
npx playwright test
```

### 1.4 查看测试报告

```bash
npx playwright show-report
```

## 三、常用命令

### 3.1 运行单个测试文件

```bash
npx playwright test tests/specs/01-login.spec.js
```

### 3.2 运行特定测试

```bash
npx playwright test -g "TC-001"
```

### 3.3 运行测试并打开浏览器（调试模式）

```bash
npx playwright test --headed
```

### 3.4 运行测试并暂停（调试模式）

```bash
npx playwright test --debug
```

### 3.5 生成测试代码（Codegen）

```bash
npx playwright codegen http://localhost:5173
```

## 四、测试配置

### 4.1 修改测试账号

编辑 `tests/config/test.config.js`：

```javascript
export const TEST_CONFIG = {
  baseURL: 'http://localhost:5173',
  apiBaseURL: 'http://localhost:3000',
  admin: {
    username: 'admin',      // 修改为你的账号
    password: 'xgsy'       // 修改为你的密码
  },
  testDataPrefix: 'test_',
  timeout: 30000
}
```

### 4.2 修改测试超时

编辑 `playwright.config.js`：

```javascript
use: {
  actionTimeout: 10000,    // 操作超时
  navigationTimeout: 30000, // 导航超时
}
```

```javascript

  test('自定义测试用例', async ({ authenticatedPage, adminLayoutPage }) => {
    // 你的测试代码
    const testData = dataGenerator.generateRoleName()
    console.log('测试数据:', testData)
    expect(true).toBeTruthy()
  })
})
```

## 六、调试技巧

### 6.1 查看测试报告

测试失败后，自动生成 HTML 报告：

```bash
npx playwright show-report
```

### 6.2 使用 Trace Viewer

1. 在 `playwright.config.js` 中启用 trace：

```javascript
use: {
  trace: 'on',  // 或 'retain-on-failure'
}
```

2. 运行测试后查看 trace：

```bash
npx playwright show-trace test-results/.../trace.zip
```

### 6.3 使用 Playwright Inspector

```bash
npx playwright test --debug
```

## 七、常见问题

### Q1: 测试提示找不到浏览器

A: 运行以下命令安装浏览器：

```bash
npx playwright install
```

### Q2: 测试超时

A: 检查开发服务器是否启动，或增加超时时间

### Q3: 元素定位失败

A: 使用 Playwright Codegen 重新生成选择器：

```bash
npx playwright codegen http://localhost:5173
```

### Q4: 如何清理测试数据

A: 测试数据使用 `test_` 前缀，可以手动删除数据库中带此前缀的数据
