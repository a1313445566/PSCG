# PSCG 智能题库系统 - WIKI

## 项目简介

PSCG (Primary School Comprehensive Quiz) 是一个面向小学教育的智能题库系统，提供学生答题、错题管理、学习分析等功能。

## 技术栈

- **前端**: Vue 3 + Element Plus + Pinia + Vite
- **后端**: Node.js + Express + MySQL
- **AI**: 支持多模型配置（OpenAI 等）

## 核心功能

### 学生端
- 学号登录（年级+班级）
- 多学科答题
- 错题巩固模式
- 学习数据统计

### 管理端
- 题目管理（CRUD + 批量导入）
- 学科和子分类管理
- 用户管理
- AI 助手（聊天 + 模型配置）
- 数据分析
