# PSCG 智能题库系统 - WIKI

## 项目简介

PSCG (Primary School Comprehensive Quiz) 是一个面向小学教育的**全栈智能题库系统**，采用 Vue3 + Element Plus + Express + MySQL 技术栈，为学生提供答题练习、错题管理、学习分析等功能，为教师/管理员提供题目管理、数据统计、AI 智能辅助、安全监控等全方位管理功能。

## 核心功能

### 学生端（14 个页面）
- 学号登录（年级 + 班级，JWT Token 认证，30 分钟会话超时）
- 多学科答题（单选题 / 多选题 / 判断题 / 阅读理解）
- 选项乱序（判断题除外，固定"对/错"）
- 答题计时 + 行为追踪
- 错题自动收集 + 巩固模式
- 学习进度跟踪 + 学习报告
- 排行榜（全局 / 学科 / TOP10）
- 答题历史查看
- 个人资料管理

### 管理端（AdminView 单页应用）
- **题目管理**：CRUD + 批量导入 + 分屏编辑 + 音频/图片附件 + 预览
- **用户管理**：学生 CRUD + 详情查看
- **学科管理**：学科/子分类/年级/班级 配置
- **AI 助手**：多模型对话 + 流式输出 + Function Calling（19 个工具）+ Token 统计
- **数据分析**：仪表盘 + 多维图表 + 错误分析 + 难度自适应
- **数据管理**：备份恢复 + 数据清理 + Excel 导出
- **安全监控**：IP 封禁/解禁 + 操作日志 + 限流状态
- **系统设置**：答题设置 + 界面名称配置

### 安全体系
- JWT 双认证（学生 + 管理员独立）
- CSRF Token 防护
- 三级速率限制（全局 / API / 提交）
- 签名缓存防重复提交
- XSS 过滤 + SQL 注入防护
- 目录遍历防护
- IP 封禁（手动 / 批量）+ 安全操作日志

## 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/a1313445566/PSCG.git
cd PSCG

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 填写数据库连接信息

# 4. 启动开发服务器（前端 + 后端）
npm run dev          # 前端 : http://localhost:5173
npm run server       # 后端 : http://localhost:3001

# 5. 或启动后端服务（生产模式）
node server.cjs      # 监听端口 3001
```
