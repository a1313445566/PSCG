## 文档分类说明

### 1. API 文档 (`/API文档/`)

存放所有 API 接口的详细文档。

| 文档 | 说明 |
|------|------|
| API.md | 完整的 RESTful API 文档，包含所有接口定义 |

**适用场景**：前后端联调、API 开发、第三方集成

### 2. 使用指南 (`/使用指南/`)

存放用户和管理员的操作指南。

| 目录/文档 | 说明 |
|----------|------|
| API使用/ | API 调用示例和说明 |
| 后台管理/ | 后台管理系统操作指南 |
| 学生端/ | 学生端使用说明 |
| 系统管理/ | 系统管理操作指南 |

**适用场景**：用户培训、日常操作参考

### 3. 开发文档 (`/开发文档/`)

存放开发规范和开发环境配置。

| 文档 | 说明 |
|------|------|
| 编码规范.md | 代码编写规范和风格指南 |
| 开发环境设置指南.md | 开发环境搭建步骤 |
| 测试指南.md | 测试策略和执行方法 |

**适用场景**：新开发者入门、代码审查

### 4. 技术文档 (`/技术文档/`)

存放技术架构、功能实现、问题排查等技术性文档。

| 文档 | 说明 |
|------|------|
| 技术文档.md | 系统整体技术架构 |
| 安全架构文档.md | 认证、签名、限流等安全设计 |
| 选项打乱功能实现说明.md | 选项随机打乱的技术实现 |
| 签名验证调试指南.md | 答题签名验证调试方法 |
| 故障排除指南.md | 常见问题诊断和解决 |

**适用场景**：系统维护、功能扩展、问题修复

### 5. 测试报告 (`/测试报告/`)

存放测试报告和测试结果。

### 6. 部署文档 (`/部署文档/`)

存放系统部署和运维相关文档。

| 文档 | 说明 |
|------|------|
| 系统部署配置指南.md | 完整的生产环境部署指南 |
| cloudflared隧道设置指南.md | 内网穿透配置 |

**适用场景**：系统部署、环境迁移

### 7. 需求文档 (`/需求文档/`)

存放功能需求和规划文档。

**适用场景**：功能规划、需求评审、验收测试

2. 文档修改需更新最后修改日期
3. 保持文档格式统一（Markdown 格式）

- 使用中文命名，便于理解
- 文件名应清晰描述内容
