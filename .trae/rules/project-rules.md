## 代码风格

### 前端代码规范
1. **Vue 组件**
   - 使用 Composition API
   - 组件命名使用 PascalCase
   - 文件名与组件名保持一致
   - 优先使用单文件组件 (.vue)
   - 合理使用 `script setup` 语法

2. **代码格式**
   - 使用 2 个空格缩进
   - 分号可选
   - 字符串使用单引号
   - 大括号使用 K&R 风格
   - 变量命名使用 camelCase
   - 常量命名使用 UPPER_SNAKE_CASE

3. **CSS 规范**
   - 使用 scoped 样式
   - 类名使用 kebab-case
   - 避免使用 ID 选择器
   - 优先使用 Element Plus 提供的类名和变量

### 后端代码规范
1. **Express 路由**
   - 路由文件使用小写字母和连字符
   - 路由处理函数使用 async/await
   - 错误处理使用 try/catch
   - 响应格式保持一致

2. **数据库操作**
   - 使用参数化查询防止 SQL 注入
   - 合理使用事务
   - 数据库连接管理

3. **缓存管理**
   - 缓存键命名规范
   - 缓存过期时间设置
   - 缓存清理策略
