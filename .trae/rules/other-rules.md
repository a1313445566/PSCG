# 其他约束和开发流程规则

## 其他约束
1. **第三方库使用**
   - 优先使用项目已有的依赖
   - 新增依赖需评估必要性
   - 版本控制

2. **代码注释**
   - 关键逻辑添加注释
   - 函数参数和返回值说明
   - 复杂算法注释

3. **文档**
   - 组件文档
   - API 文档
   - 项目文档

## 开发流程
1. **分支管理**
   - 主分支 (main)
   - 开发分支 (develop)
   - 特性分支 (feature/*)
   - 修复分支 (fix/*)

2. **提交规范**
   - 提交信息清晰明了
   - 关联 issue
   - 代码审查

3. **代码合并**
   - 代码审查通过后合并
   - 测试通过后部署
