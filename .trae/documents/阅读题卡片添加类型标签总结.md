## 设计说明

### 颜色方案

**阅读题标签**：
- 渐变：`linear-gradient(135deg, #f093fb 0%, #f5576c 100%)`
- 粉红色系，与阅读题的温馨、柔和特点相符

**其他题目类型**（参考 QuestionCard.vue）：
- 默认渐变：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 可根据不同题型扩展不同颜色

### 布局调整

**修改前**：
```
[图标] 阅读材料
```

**修改后**：
```
[图标] 阅读材料        [阅读理解]
← header-left ───────→  ← 标签 →
```

使用 `justify-content: space-between` 实现左右分布

## 样式特性

1. **圆角设计**：`border-radius: 12px`，与整体 UI 风格一致
2. **渐变背景**：使用粉红色渐变，区分于其他题型
3. **文字阴影**：`text-shadow: 0 1px 2px rgba(0, 0, 0, 0.2)`，增强可读性
4. **响应式**：`flex-shrink: 0`，防止标签被压缩
5. **间距合理**：`padding: 0.3rem 0.8rem`，视觉舒适
